# Valve's new Steam Machine verification system is silent on these Steam Deck-busters

**Valve 全新的 Steam Machine 验证系统对这些“Steam Deck 杀手级”游戏保持沉默**

About a month ago, Valve announced that it would expand its long-standing Steam Deck Verified program to the now-shipping Steam Machine, offering a separate rating of Steam games’ compatibility and playability for the fresh living room-focused hardware.

大约一个月前，Valve 宣布将把长期运行的“Steam Deck 验证”计划扩展到现已发货的 Steam Machine 上，为这款全新的客厅专用硬件提供一套独立的 Steam 游戏兼容性和可玩性评级。

Now that those ratings started appearing on Steam store pages last night (under a “Learn More” link next to Steam Deck Compatibility), we’ve found that Valve is frustratingly “still learning about” Steam Machine compatibility for dozens of games that the Steam Deck is too weak to run capably.

随着这些评级昨晚开始出现在 Steam 商店页面上（位于“Steam Deck 兼容性”旁边的“了解更多”链接下），我们沮丧地发现，对于数十款 Steam Deck 性能不足以流畅运行的游戏，Valve 对其 Steam Machine 的兼容性状态仍处于“仍在了解中”的阶段。

The Steam Machine compatibility for many Steam games is pretty simple to figure out, of course. If a game is already verified on Steam Deck, it is seemingly guaranteed to be verified on the Steam Machine, as far as we can tell. On the other side, games that have already been confirmed not to work with SteamOS (which can happen for various reasons) obviously won’t work on the SteamOS-powered Steam Machine.

当然，许多 Steam 游戏的 Steam Machine 兼容性很容易判断。据我们所知，如果一款游戏已经在 Steam Deck 上获得验证，那么它在 Steam Machine 上几乎肯定也会获得验证。反之，那些已被确认无法在 SteamOS 上运行的游戏（原因多种多样），显然也无法在搭载 SteamOS 的 Steam Machine 上运行。

The messy middle is where a Steam Machine Verified badge could come in most handy. These are games that Valve has confirmed will load on SteamOS, but which the aging, portable Steam Deck can’t handle at the 1200×800, 30 fps standard that Valve requires at default settings (for the Steam Machine, this requirement grows to 1080p and 30 fps).

最令人困惑的中间地带，正是“Steam Machine 验证”徽章最能发挥作用的地方。这些游戏是 Valve 已确认可以在 SteamOS 上加载，但老旧的便携式 Steam Deck 无法以 Valve 要求的默认设置（1200×800 分辨率、30 fps）运行的游戏（对于 Steam Machine，这一要求提高到了 1080p 和 30 fps）。

On the Steam Store, these games show up as “Unsupported” on Steam Deck because “the game’s graphics settings cannot be configured to run well on Steam Deck” or “this game requires manual configuration of graphics settings to perform well on Steam Deck.” That is frankly unsurprising.

在 Steam 商店中，这些游戏在 Steam Deck 上显示为“不支持”，原因是“无法配置游戏的图形设置以在 Steam Deck 上良好运行”或“该游戏需要手动配置图形设置才能在 Steam Deck 上良好运行”。坦率地说，这并不令人意外。

Thus far, every such graphically unsupported Steam Deck game we’ve found on the Steam store is currently listed with an “Unknown” Steam Machine compatibility status. For all of these titles, Valve merely says it is “still learning about” the game and that “we do not currently have further information regarding Steam Machine compatibility.”

到目前为止，我们在 Steam 商店中发现的每一款此类图形性能不支持的 Steam Deck 游戏，其 Steam Machine 兼容性状态目前都显示为“未知”。对于所有这些游戏，Valve 仅表示“仍在了解中”，并称“我们目前没有关于 Steam Machine 兼容性的进一步信息”。

That’s a shame, because many of these games that struggle on the Steam Deck can no doubt run just fine on the much more powerful Steam Machine hardware. And new Steam Machine owners will surely want to know whether their fresh living room box can play these games, many of which have appeared on Steam’s Top Sellers list (either currently or in the past).

这很遗憾，因为许多在 Steam Deck 上运行吃力的游戏，无疑可以在性能更强大的 Steam Machine 硬件上流畅运行。新的 Steam Machine 用户肯定想知道他们这台全新的客厅设备能否运行这些游戏，其中许多游戏都曾（或正在）出现在 Steam 的畅销榜单上。

There is some good news for Steam Machine users in last night’s Verification update, though. Games with text that is too small to read on the Steam Deck are perfectly fine when played on a big TV. That means Steam Deck “Playable” games like *007: First Light* and *Lies of P* get bumped up to “Verified” status on the Steam Machine.

不过，昨晚的验证更新对 Steam Machine 用户来说也有一些好消息。在 Steam Deck 上因字体太小而难以阅读的游戏，在电视大屏上游玩时完全没有问题。这意味着像《007: First Light》和《匹诺曹的谎言》(Lies of P) 这样在 Steam Deck 上仅为“可玩”的游戏，在 Steam Machine 上被提升到了“已验证”状态。

But games that Valve bumps down to “Playable” on Steam Deck for requiring an on-screen keyboard still get a similar warning on the Steam Machine, so you might want to keep a wireless keyboard nearby on your couch.

但那些因为需要屏幕键盘而在 Steam Deck 上被降级为“可玩”的游戏，在 Steam Machine 上依然会收到类似的警告，所以你可能需要在沙发旁备一个无线键盘。

We’ve included a list of nearly two dozen games we’ve found that Valve says can run on SteamOS but can’t run well on Steam Deck, and that Valve has yet to test on the Steam Machine. Hopefully they’ll get the helpful up or down Steam Machine rating they deserve soon.

我们列出了近二十款我们发现的游戏，Valve 表示它们可以在 SteamOS 上运行，但在 Steam Deck 上表现不佳，且 Valve 尚未在 Steam Machine 上进行测试。希望它们能尽快获得应有的、有参考价值的 Steam Machine 评级。
