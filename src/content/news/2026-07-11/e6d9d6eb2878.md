# Towards the Explainability of Temporal Graph Networks via Memory Backtracking and Topological Attribution
# 通过记忆回溯与拓扑归因实现时序图网络的可解释性

**Abstract:** Temporal graphs are ubiquitous in real-world applications and Temporal Graph Networks (TGNs) have achieved superior predictive accuracy. Understanding which historical events drive model predictions can enhance trustworthiness of TGNs. 

**摘要：** 时序图在现实世界的应用中无处不在，而时序图网络（TGNs）已经实现了卓越的预测精度。理解哪些历史事件驱动了模型预测，可以增强 TGNs 的可信度。

Existing explanation methods overlook the memory module, the core component that records and updates node histories, leaving the influence of past events unexplored. To address this, we attribute TGNs predictions through the topology attribution tree and memory backtracking tree. 

现有的解释方法忽略了记忆模块——即记录和更新节点历史的核心组件，导致过去事件的影响未被探索。为了解决这个问题，我们通过拓扑归因树和记忆回溯树对 TGNs 的预测进行归因。

The topology attribution tree captures the influence of neighbors and their memory vectors, then the memory backtracking tree quantifies how historical events shape node memory vectors. We apply the LRP in TGNs, ensuring that the total contribution of events equals the logits of model. 

拓扑归因树捕捉邻居及其记忆向量的影响，随后记忆回溯树量化历史事件如何塑造节点记忆向量。我们将层级相关性传播（LRP）应用于 TGNs，确保事件的总贡献等于模型的逻辑值（logits）。

Finally, top-k selection may be unfaithful due to the nonlinear mapping from logits to probabilities, we design optimization objectives to identify the important events. Experiments on nine temporal graph datasets, spanning node property prediction, link prediction tasks and graph classification tasks, show that our method provides faithful explanations and outperforms state-of-the-art baselines. 

最后，由于从逻辑值到概率的非线性映射，Top-k 选择可能不够忠实，因此我们设计了优化目标来识别重要事件。在涵盖节点属性预测、链路预测任务和图分类任务的九个时序图数据集上的实验表明，我们的方法提供了忠实的解释，并优于现有的最先进基准模型。
