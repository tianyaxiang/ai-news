# SHIFT: Survival Prediction from Incomplete and Heterogeneous Genomic Data
# SHIFT：基于不完整且异构基因组数据的生存预测

Genomic prediction models often fail to transfer across institutions because sequencing panels differ across sites, creating structural feature missingness at deployment. Existing approaches to this challenge typically restrict analysis to genes shared across cohorts, exclude patients with incomplete profiles, or rely on test-time imputation, all of which can reduce robustness and limit the use of multi-center data.

基因组预测模型往往难以在不同机构间迁移，因为各站点的测序面板存在差异，导致在部署时出现结构性特征缺失。针对这一挑战，现有方法通常仅限于分析各队列共有的基因、排除具有不完整配置文件的患者，或依赖测试时的插补（imputation），而这些做法都会降低模型的鲁棒性并限制多中心数据的使用。

We propose Survival prediction Handling Incomplete Features using Transformer (SHIFT), a missingness-aware survival model that directly predicts from incomplete genomic inputs without test-time imputation. SHIFT represents each genomic feature separately and uses masked self-attention, along with a feature-availability mask, so that predictions are based only on observed inputs.

我们提出了“基于 Transformer 处理不完整特征的生存预测”（SHIFT）模型，这是一种具备缺失感知能力的生存模型，无需测试时插补即可直接从不完整的基因组输入中进行预测。SHIFT 将每个基因组特征单独表示，并结合掩码自注意力机制（masked self-attention）和特征可用性掩码，确保预测仅基于观测到的输入。

Further, we introduce variable-rate feature masking during training to improve robustness to heterogeneous missingness patterns. We evaluate the approach on glioblastoma and lung squamous cell carcinoma with external validation across multiple cohorts, including a challenging setting with severe cross-cohort panel mismatch.

此外，我们在训练过程中引入了变速率特征掩码，以提高模型对异构缺失模式的鲁棒性。我们在胶质母细胞瘤和肺鳞状细胞癌数据集上评估了该方法，并跨多个队列进行了外部验证，其中包括存在严重跨队列面板不匹配的挑战性场景。

Across these settings, SHIFT shows strong generalization and compares favorably with standard survival baselines and imputation-based approaches, while using a single model across differing feature sets. We also find that incorporating patients from incomplete cohorts during development can improve performance on external data, suggesting that partially observed cohorts need not be excluded from model building.

在这些场景中，SHIFT 表现出了强大的泛化能力，且优于标准的生存基准模型和基于插补的方法，同时能够使用单一模型处理不同的特征集。我们还发现，在开发过程中纳入来自不完整队列的患者可以提高模型在外部数据上的表现，这表明在模型构建过程中无需排除部分观测到的队列。

These results support missingness-aware modeling as a practical strategy for multi-center survival prediction in precision oncology.

这些结果证明，缺失感知建模是精准肿瘤学中多中心生存预测的一种实用策略。
