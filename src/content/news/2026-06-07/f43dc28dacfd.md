# sveltejs / svelte

**What is Svelte? Svelte is a new way to build web applications. It's a compiler that takes your declarative components and converts them into efficient JavaScript that surgically updates the DOM. Learn more at the Svelte website, or stop by the Discord chatroom.**

什么是 Svelte？Svelte 是一种构建 Web 应用程序的新方法。它是一个编译器，能够将你的声明式组件转换为高效的 JavaScript，从而对 DOM 进行精准更新。欲了解更多信息，请访问 Svelte 官网或加入我们的 Discord 聊天室。

**Supporting Svelte**
**Svelte is an MIT-licensed open source project with its ongoing development made possible entirely by fantastic volunteers. If you'd like to support their efforts, please consider: Becoming a backer on Open Collective. Funds donated via Open Collective will be used for compensating expenses related to Svelte's development such as hosting costs. If sufficient donations are received, funds may also be used to support Svelte's development more directly.**

支持 Svelte
Svelte 是一个采用 MIT 许可协议的开源项目，其持续开发完全得益于出色的志愿者们。如果你想支持他们的工作，请考虑：在 Open Collective 上成为赞助者。通过 Open Collective 捐赠的资金将用于补偿与 Svelte 开发相关的费用，例如托管成本。如果收到的捐款充足，资金也可能被直接用于支持 Svelte 的开发。
