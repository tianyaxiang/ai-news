# UniVL: Unified Vision-Language Embedding for Spatially Grounded Contextual Image Generation
# UniVL：用于空间定位上下文图像生成的统一视觉-语言嵌入

**Abstract:** We introduce spatially grounded contextual image generation, a controllable image generation task that reframes the conditioning paradigm. Instead of supplying a reference image and a global text prompt through two separate encoders, one for vision and one for language, UniVL is trained to bind semantics to spatial locations directly from a single unified visual input, where the textual instruction is rendered onto the spatial mask. This removes the need for a standalone text encoder at inference time. The resulting model supports contextual image generation by following user-specified instructions about what should appear where, while substantially reducing computation.

**摘要：** 我们引入了“空间定位上下文图像生成”（spatially grounded contextual image generation），这是一种重构了条件生成范式的可控图像生成任务。UniVL 不再通过视觉和语言两个独立的编码器分别提供参考图像和全局文本提示，而是通过训练将语义直接绑定到单一统一视觉输入中的空间位置上，其中文本指令被渲染在空间掩码（spatial mask）上。这消除了推理阶段对独立文本编码器的需求。该模型通过遵循用户关于“什么内容应出现在何处”的指定指令来支持上下文图像生成，同时大幅降低了计算量。

To address this task, we propose a framework in which the UniVL encoder, adapted from an optical-character-recognition-pretrained backbone, reads the unified condition optically and produces a UniVL embedding, fVIL, that fuses visual and semantic intent with spatial locations in a single token sequence. A two-stage pipeline first aligns UniVL with the VAE embedding space and then conditions a pretrained diffusion backbone entirely on UniVL embeddings, eliminating the standalone text encoder, such as T5. Although this reframing uses a deliberately minimal text interface, it yields strong empirical gains.

为了解决这一任务，我们提出了一个框架：UniVL 编码器改编自预训练的光学字符识别（OCR）主干网络，它通过光学方式读取统一条件，并生成 UniVL 嵌入（fVIL），将视觉和语义意图与空间位置融合在单一的标记序列中。该两阶段流水线首先将 UniVL 与 VAE 嵌入空间对齐，然后完全基于 UniVL 嵌入对预训练的扩散模型主干进行条件化，从而消除了诸如 T5 之类的独立文本编码器。尽管这种重构采用了刻意简化的文本接口，但它在实验中取得了显著的性能提升。

On UniVL-ImgGen, a benchmark of 477K mask-annotated images that we construct for training and evaluation, UniVL improves image quality over text-prompted baselines, reducing FID from 14 to 11 and increasing PSNR from 16 to 20. It also eliminates the text encoder entirely, reducing inference TFLOPs by up to 52% and runtime by up to 44%. Additional ablation studies validate the contributions of the proposed components, paving the way for efficient, spatially grounded image generation with a unified conditioning paradigm.

在 UniVL-ImgGen（我们为训练和评估构建的一个包含 477K 张掩码标注图像的基准测试集）上，UniVL 在图像质量方面优于基于文本提示的基准模型，将 FID 从 14 降低至 11，并将 PSNR 从 16 提高至 20。此外，它完全消除了文本编码器，使推理 TFLOPs 降低了高达 52%，运行时间缩短了高达 44%。额外的消融研究验证了所提组件的贡献，为采用统一条件生成范式实现高效、空间定位的图像生成铺平了道路。
