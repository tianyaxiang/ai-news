### Odoo / Odoo

**Odoo is a suite of web-based open-source business apps. The main Odoo Apps include an Open Source CRM, Website Builder, eCommerce, Warehouse Management, Project Management, Billing & Accounting, Point of Sale, Human Resources, Marketing, Manufacturing, and more.**
Odoo 是一套基于 Web 的开源商业应用程序套件。主要的 Odoo 应用包括开源 CRM、网站构建器、电子商务、仓库管理、项目管理、账单与会计、销售点 (POS)、人力资源、营销、制造等。

**Odoo Apps can be used as stand-alone applications, but they also integrate seamlessly so you get a full-featured Open Source ERP when you install several Apps.**
Odoo 应用既可以作为独立的应用程序使用，也可以无缝集成；因此，当您安装多个应用时，便可获得一套功能齐全的开源 ERP 系统。

**Getting started with Odoo: For a standard installation, please follow the Setup instructions from the documentation. To learn the software, we recommend the Odoo eLearning, or Scale-up, the business game. Developers can start with the developer tutorials.**
Odoo 入门：如需进行标准安装，请遵循文档中的设置说明。若要学习该软件，我们推荐使用 Odoo 在线学习平台 (eLearning) 或商业游戏“Scale-up”。开发人员可以从开发者教程开始学习。

**Security: If you believe you have found a security issue, check our Responsible Disclosure page for details and get in touch with us via email.**
安全性：如果您认为自己发现了安全问题，请查看我们的“负责任披露”(Responsible Disclosure) 页面了解详情，并通过电子邮件与我们联系。
