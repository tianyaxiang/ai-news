### pumaDB

**Launched this week, a small hosted memory layer for AI agents.**
本周发布：一个专为 AI 智能体设计的轻量级托管内存层。

**Most AI agent workflows lose useful context between sessions, tools, and chats.**
大多数 AI 智能体的工作流在不同的会话、工具和聊天之间会丢失有用的上下文信息。

**The usual fixes are either too manual, like copying notes into docs, or too heavy, like setting up a database, vector store, or custom RAG stack.**
常见的解决方案要么过于繁琐（例如手动将笔记复制到文档中），要么过于沉重（例如需要配置数据库、向量存储或自定义 RAG 技术栈）。

**pumaDB gives agents a simple shared place to save and reuse notes, facts, preferences, project context, transcripts, task state, and other useful memory.**
pumaDB 为智能体提供了一个简单的共享空间，用于保存和重用笔记、事实、偏好设置、项目上下文、转录内容、任务状态以及其他有用的记忆信息。

**No database setup, vector DB, or infrastructure to manage.**
无需配置数据库、向量数据库，也无需管理任何基础设施。
