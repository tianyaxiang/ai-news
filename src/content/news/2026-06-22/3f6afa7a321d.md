# Cloudback MCP Server

**Cloudback**
Automatic backups and restores for Git and Linear.
Cloudback 为 Git 和 Linear 提供自动备份与恢复服务。

**Engineering & Development**
Automatic daily backups and restores for GitHub, GitLab, Azure DevOps, and Linear. Git backups capture repositories, metadata, and LFS; Linear backups capture the whole workspace - issues, projects, cycles, documents, and more. Store them in Cloudback Managed Storage or customer-managed storage (Amazon S3, Azure Blob, Google Cloud, Wasabi, and more). AES-256 encryption with customer-managed keys (RSA Lockbox). SOC 2 Type II compliant. 1.7k+ customers, 14k+ repositories, 4M+ backups.
工程与开发：为 GitHub、GitLab、Azure DevOps 和 Linear 提供每日自动备份与恢复。Git 备份涵盖代码仓库、元数据和 LFS；Linear 备份则涵盖整个工作区，包括议题（Issues）、项目、周期、文档等。数据可存储在 Cloudback 托管存储或客户自管存储（如 Amazon S3、Azure Blob、Google Cloud、Wasabi 等）中。支持使用客户自管密钥（RSA Lockbox）进行 AES-256 加密，并符合 SOC 2 Type II 标准。目前拥有 1700 多家客户，管理超过 1.4 万个仓库，累计完成 400 万次备份。

**Cloudback MCP Server**
Launching today: Manage your backups from Claude, Cursor, and VS Code.
Cloudback now ships an official MCP server. Connect Claude Code, Claude Desktop, Cursor, VS Code, or any MCP client and manage your backup definitions, schedules, storages, and retention policies through plain-language chat - across GitHub, GitLab, Azure DevOps, and Linear. One prompt can reschedule and re-retain 300 backup definitions at once - no dashboard clicks, no throwaway scripts. Runs as a Docker image (myrtlelabs/cloudback-mcp).
Cloudback MCP Server 今日发布：通过 Claude、Cursor 和 VS Code 管理你的备份。
Cloudback 现已推出官方 MCP 服务器。连接 Claude Code、Claude Desktop、Cursor、VS Code 或任何 MCP 客户端，即可通过自然语言对话管理 GitHub、GitLab、Azure DevOps 和 Linear 的备份定义、计划、存储和保留策略。只需一个提示词，即可同时重新调度和设置 300 个备份定义，无需点击仪表板，也无需编写一次性脚本。该服务以 Docker 镜像（myrtlelabs/cloudback-mcp）形式运行。
