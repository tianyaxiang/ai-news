# Firefox 153 available with support for Vulkan video decoding, JPEG-XL
# Firefox 153 发布，支持 Vulkan 视频解码及 JPEG-XL

**Firefox 153 Available With Support For Vulkan Video Decoding, Experimental JPEG-XL**
**Firefox 153 发布，支持 Vulkan 视频解码及实验性 JPEG-XL**

Mozilla has uploaded the Firefox 153.0 release binaries today for the newest monthly update to this cross-platform web browser. Making Firefox 153 exciting is that it's the newest Extended Support Release "ESR" as well as introducing Vulkan Video decode support.
Mozilla 今日上传了 Firefox 153.0 的正式版二进制文件，这是该跨平台网页浏览器的最新月度更新。Firefox 153 的亮点在于它不仅是全新的长期支持版本（ESR），还引入了对 Vulkan 视频解码的支持。

Firefox 153 is the newest ESR release designed for enterprises, schools, and other organization for long-term stability. On the feature side, Firefox 153 introduces Vulkan Video decoding support.
Firefox 153 是专为企业、学校及其他机构设计的最新 ESR 版本，旨在提供长期稳定性。在功能方面，Firefox 153 引入了 Vulkan 视频解码支持。

Firefox on Linux to date has focused on the Video Acceleration API (VA-API) that works well for some vendors but lacks official support from NVIDIA unless using the independent NVIDIA-VAAPI driver project that layers VA-API on NVDEC. There are also various embedded drivers and more not supporting VA-API. Now with Firefox there is initial Vulkan Video decode support for that modern, cross-vendor and cross-platform video acceleration interface.
长期以来，Linux 上的 Firefox 主要依赖视频加速 API (VA-API)。虽然该接口在某些厂商的硬件上表现良好，但缺乏 NVIDIA 的官方支持，除非用户使用将 VA-API 叠加在 NVDEC 之上的第三方 NVIDIA-VAAPI 驱动项目。此外，还有许多嵌入式驱动程序并不支持 VA-API。现在，Firefox 提供了对 Vulkan Video 的初步解码支持，这是一种现代化的、跨厂商且跨平台的视频加速接口。

Firefox 153 also brings a number of PDF enhancements, continued enablement around JPEG-XL, HDR video playback on Windows, and more. Within the Firefox Labs area is where the experimental support for the JPEG-XL image format can be enabled.
Firefox 153 还带来了多项 PDF 增强功能、对 JPEG-XL 的持续支持、Windows 上的 HDR 视频播放等改进。用户可以在“Firefox Labs”（Firefox 实验室）区域开启对 JPEG-XL 图像格式的实验性支持。

More details within the beta release notes. Firefox 153 release binaries can be downloaded from ftp.mozilla.org.
更多详细信息请参阅测试版发布说明。Firefox 153 的正式版二进制文件可从 ftp.mozilla.org 下载。
