# ReactOS "Open-Source Windows" Project Now Capable Of Running Half-Life 2
# ReactOS “开源 Windows” 项目现已能够运行《半条命 2》

ReactOS "Open-Source Windows" Project Now Capable Of Running Half-Life 2 Written by Michael Larabel in Operating Systems on 5 July 2026 at 06:42 AM EDT.
ReactOS “开源 Windows” 项目现已能够运行《半条命 2》，由 Michael Larabel 撰写，发布于 2026 年 7 月 5 日美国东部时间上午 06:42，归类于操作系统板块。

One month ago it was exciting to see the open-source ReactOS operating system running Valve's Half-Life game. Little to realize less than 30 days later it would also be running Half-Life 2.
一个月前，看到开源操作系统 ReactOS 能够运行 Valve 的《半条命》游戏令人感到兴奋。没想到不到 30 天后，它竟然也能运行《半条命 2》了。

In early June, ReactOS as this "open-source Windows" platform implementing Windows software game/app and driver binary compatibility was running Half-Life. It was demoed with a Intel Sandy Bridge era desktop using NVIDIA GeForce 8400GS graphics.
今年 6 月初，作为实现 Windows 软件游戏/应用及驱动程序二进制兼容性的“开源 Windows”平台，ReactOS 成功运行了《半条命》。当时的演示是在一台采用英特尔 Sandy Bridge 时代处理器和 NVIDIA GeForce 8400GS 显卡的台式机上进行的。

The ReactOS project shared on X this weekend that the newer Half-Life 2 game is now up and running as well on this three decade old free software project.
ReactOS 项目组本周末在 X（原 Twitter）上分享称，这款已有三十年历史的自由软件项目，现在也已经能够运行更新的《半条命 2》游戏了。

With a ReactOS nightly build, Half-Life 2 successfully ran with the game appearing to work nicely in-game. That user testing was done with a GeForce GTX 960 graphics card when installing the NVIDIA 368.61 legacy Windows driver. Plus installing Creative Sound Blaster Audigy Windows drivers too.
通过 ReactOS 的每日构建版本，《半条命 2》成功运行，且游戏内表现良好。该用户测试是在安装了 NVIDIA 368.61 旧版 Windows 驱动程序的 GeForce GTX 960 显卡上完成的，同时还安装了 Creative Sound Blaster Audigy 的 Windows 驱动程序。

Separately, this week also saw ReactOS introducing its first Windows NT6 system call as the first baby step toward Windows Vista and newer compatibility.
另外，本周 ReactOS 还引入了其首个 Windows NT6 系统调用，这是迈向兼容 Windows Vista 及更新版本系统的第一小步。
