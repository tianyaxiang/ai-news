# Corvorum OS Server Edition

**Corvorum OS Server Edition 1.0: A server operating system with local AI and a ready-to-use environment from the first boot.**

Corvorum OS Server Edition 1.0：一款具备本地 AI 功能，且开箱即用的服务器操作系统。

Recently, I introduced Corvorum OS 1.0 Desktop Edition, a minimal Ubuntu-based Linux distro designed for developers who want a modern, functional environment with its own identity, as well as for Windows users looking for a change. Now, I present its server version, which features the same AI integrated with Ollama, Python, Docker, Ansible, and Kubernetes k3s, ready to set up a minimal DevOps infrastructure. It includes tools like `mc` and `btop` to comfortably manage files from the CLI and monitor processes. You can also work with Ollama using its native command and different AIs, or use the `ai` command, which invokes the Odin-assist model based on Qwen2.5:1.5B.

最近，我发布了 Corvorum OS 1.0 桌面版，这是一个基于极简 Ubuntu 的 Linux 发行版，专为追求现代感、功能性及独特个性的开发者，以及寻求系统迁移的 Windows 用户而设计。现在，我带来了它的服务器版本，该版本集成了与桌面版相同的 AI 功能（基于 Ollama），并预装了 Python、Docker、Ansible 和 Kubernetes k3s，可直接用于搭建极简的 DevOps 基础设施。系统内置了 `mc` 和 `btop` 等工具，方便用户通过命令行管理文件及监控进程。此外，你既可以使用 Ollama 原生命令调用不同的 AI 模型，也可以直接运行 `ai` 命令，调用基于 Qwen2.5:1.5B 的 Odin-assist 模型。

**What does version 1.0 include?**

**1.0 版本包含哪些内容？**

**Terminal with its own identity:** Upon opening the terminal, you will see the Corvorum OS banner with system information: kernel version, available local AI models, and the main commands ready to use.

**具有独特标识的终端：** 打开终端时，你将看到 Corvorum OS 的横幅信息，包括系统内核版本、可用的本地 AI 模型以及主要命令的使用指南。

```text

Local AI: 3 model(s) available
Type:
- 'ai' to consult the Odin assistant
- 'btop' for task manager
- 'ctop' for docker container management
- 'mc' to run the file explorer
Corvorum OS on k3s: default
Transmuting ›
```

**Project Philosophy:** Corvorum OS was born from a simple belief: regain control of what you do and stop depending on third parties. We live in an era of subscription-based Artificial Intelligence and systems that implement cloud-based AI. This creates a dependency that we can still sustain, but which will become more expensive and difficult to manage in the future. For that moment, I want to offer this alternative—already mature and independent—that can save you from paying high prices for AI computing or having your computer usage telemetry tracked. That is why the system integrates local AI, Windows support, and a development environment as part of the system. We don't know exactly what the future holds, but what we can and must do is build it through the present.

**项目理念：** Corvorum OS 源于一个简单的信念：夺回对自己所作所为的控制权，不再依赖第三方。我们生活在一个充斥着订阅制人工智能和云端 AI 系统的时代。这产生了一种我们目前尚能维持，但未来将变得更加昂贵且难以管理的依赖关系。为了应对未来，我希望提供这样一个成熟且独立的替代方案，让你免于支付高昂的 AI 计算费用，也不必担心电脑使用行为被遥测。因此，该系统将本地 AI、Windows 支持和开发环境整合为一体。我们无法确切预知未来，但我们能够且必须通过当下构建未来。

**How to test it:** The Corvorum OS 1.0 Server Edition ISO image is available for download: 👉 [https://sourceforge.net/projects/corvorum-os/files/release1.0/corvorum-os-se_1.0_amd64.iso/download] You can test it in a virtual machine (VirtualBox, VMware, QEMU) or install it directly on compatible hardware (amd64 architecture). If you wish to install it, the credentials are: User: `scholaris`, Password: `frater`. These can be changed during the CLI installation process performed with `penguin's eggs`.

**如何测试：** Corvorum OS 1.0 服务器版的 ISO 镜像现已开放下载：👉 [https://sourceforge.net/projects/corvorum-os/files/release1.0/corvorum-os-se_1.0_amd64.iso/download]。你可以在虚拟机（VirtualBox, VMware, QEMU）中进行测试，或直接安装在兼容的硬件（amd64 架构）上。安装凭据为：用户名 `scholaris`，密码 `frater`。这些信息可以在使用 `penguin's eggs` 进行命令行安装的过程中进行修改。

**Recommended minimum requirements:** 8 GB RAM (8 GB recommended for local AI models), 20 GB disk space, 64-bit CPU, Intel Iris XE GPU or higher.

**推荐最低配置：** 8 GB 内存（本地 AI 模型建议 8 GB 以上）、20 GB 磁盘空间、64 位 CPU、Intel Iris XE 或更高级别的 GPU。

**Roadmap:** Corvorum OS 1.0 is the starting point. The project has a broader ecosystem in development:
*   **Odin Agent:** Odin assistant that allows file searching, internet searching, document creation, and other functionalities.
*   **Update Notification:** System that checks official repositories.
*   **AI Agent Configuration:** Tool to check equipment requirements and choose the best default agents for each machine, also giving the option to change them manually without having to search for them in Ollama.
*   **Community Repository:** Opening soon for external contributions, with a progressive review system and the Technomantus Edition as an official curated branch.

**路线图：** Corvorum OS 1.0 只是一个起点。该项目目前正在开发更广泛的生态系统：
*   **Odin Agent：** Odin 助手，支持文件搜索、互联网搜索、文档创建及其他功能。
*   **更新通知：** 用于检查官方仓库的系统。
*   **AI 代理配置：** 用于检查设备需求并为每台机器选择最佳默认代理的工具，同时也支持手动更改，无需在 Ollama 中手动搜索。
*   **社区仓库：** 即将开放外部贡献，采用渐进式审核系统，并将 Technomantus 版本作为官方精选分支。
