### Correcting Performance Estimation Bias in Imbalanced Classification with Minority Subconcepts
### 纠正不平衡分类中少数子概念的性能估计偏差

**Abstract:** Class-level evaluation can conceal substantial performance disparities across subconcepts within the same class, causing models that perform well on average to fail on specific subpopulations.
**摘要：** 类级评估可能会掩盖同一类别内不同子概念之间巨大的性能差异，导致在平均水平上表现良好的模型在特定子群体上却表现不佳。

Prior work has shown that common evaluation measures for imbalanced classification are biased toward larger minority subconcepts and that utility-based reweighting using true subconcept labels can mitigate this bias; however, such labels are rarely available at test time.
先前的研究表明，用于不平衡分类的常用评估指标往往偏向于较大的少数子概念，而使用真实子概念标签进行基于效用的重加权可以缓解这种偏差；然而，在测试阶段通常无法获得这些标签。

We introduce a practical utility-weighted evaluation that replaces unavailable subconcept labels with predicted posterior probabilities from a multiclass subconcept model.
我们引入了一种实用的效用加权评估方法，该方法利用多类子概念模型预测的后验概率来替代无法获取的子概念标签。

Evaluation weights are defined as the expected utility under this posterior, yielding a soft, uncertainty-aware metric we call predicted-weighted balanced accuracy (pBA).
评估权重定义为该后验概率下的期望效用，从而产生了一种我们称之为“预测加权平衡准确率”（pBA）的软性、具备不确定性感知能力的指标。

Experiments on tabular benchmarks as well as medical-imaging and text datasets show that unweighted scores can be misleading under within-class heterogeneity, while pBA provides more stable and interpretable assessments when subconcept distributions are uneven but not pathological.
在表格基准测试以及医学影像和文本数据集上的实验表明，在类内异质性存在的情况下，未加权的分数可能会产生误导；而当子概念分布不均匀但未达到病态程度时，pBA 能提供更稳定且可解释的评估结果。
