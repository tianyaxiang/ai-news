1 How to draw To draw you need two things. Something to draw with and something to draw on. The natural inclination might be to use a real device that you can touch with a pen and produce lines, but there are worse and less convenient ways to draw with 9front that you might prefer. While this document is written with a computer one can draw on in mind, most of it is applicable to any of the input methods below. Maybe you can think of new ones.
1 如何绘画。要绘画，你需要两样东西：用来画画的工具和用来画画的载体。人们的自然倾向可能是使用一种真实的设备，可以用笔触碰并产生线条，但使用 9front 绘画还有更糟糕、更不方便的方法，或许你反而会喜欢。虽然本文档是针对可以在上面绘画的计算机编写的，但其中大部分内容适用于以下任何输入方法。也许你能想到新的方法。

1.1 The Machine 1.1.1 A Computer If you don't care about drawing on screens, you can learn How to draw on any computer running 9front.
1.1 机器 1.1.1 计算机。如果你不在乎在屏幕上绘画，你可以在任何运行 9front 的计算机上学习“如何绘画”。

1.1.2 Two in One All in one. Canvas and brush - perhaps the ideal choice for those who like to draw on paper. Wacomized laptops with pens. Crack the screen open and draw on it. Here is a list of devices proven to work well for the purposes of How to draw: ThinkPad X41t, Thinkpad X60t, X61t, ThinkPad X380 Yoga, ThinkPad X1 Yoga 3rd Gen, ThinkPad X1 Titanium Yoga. There's probably a million other 2in1 9front art machines that work well, you just have to find them, and preferably tell someone you did.
1.1.2 二合一设备。一体化。画布和画笔——对于那些喜欢在纸上绘画的人来说，这可能是理想的选择。带有触控笔的 Wacom 化笔记本电脑。打开屏幕，在上面绘画。以下是已被证明非常适合“如何绘画”用途的设备列表：ThinkPad X41t、Thinkpad X60t、X61t、ThinkPad X380 Yoga、ThinkPad X1 Yoga 第三代、ThinkPad X1 Titanium Yoga。可能还有一百万种运行良好的二合一 9front 艺术机器，你只需要找到它们，最好告诉别人你找到了。

1.2 The Input 1.2.1 Mouse Mouse drawing is a very old technique. Favored for its compatibility with any computer that has a mouse.
1.2 输入方式 1.2.1 鼠标。鼠标绘画是一种非常古老的技术。因为它与任何带有鼠标的计算机兼容而受到青睐。

1.2.2 Finger Touching the screen emulates left mouse clicks. There's no way to emulate any other, so a mouse needs to be kept nearby. Some people's entire gimmick depends on the fact that they finger paint their art. If you have a device that allows it, you can finger paint with 9front.
1.2.2 手指。触摸屏幕模拟鼠标左键点击。没有办法模拟其他按键，所以需要随身携带鼠标。有些人的全部噱头都依赖于他们用手指绘画这一事实。如果你有允许这样做的设备，你就可以用 9front 进行手指绘画。

1.2.3 Tablet External tablets with screens will (probably) not work with 9front. External tablets without screens might work. Similarly to 2in1 drawing computers, external tablet pens with buttons can potentially simulate all three mouse clicks. I've seen it. Of the three tablets that I personally tried, the one to recommend is GAOMON S620. Bought specifically for 9front, I ended up drawing the bulk of early Techno-Mage with it in OpenBSD. It still works great in 9front. Plug n play. Not so much in OpenBSD anymore. The other two: Acts like a touchpad (relative positioning) Wacom CTL-672 Does not work Wacom CTL-490. Some more information on tablets: [fqa] 3.2.4 - Tablet Digitizers
1.2.3 数位板。带屏幕的外接数位板（可能）无法在 9front 上工作。不带屏幕的外接数位板可能可以。与二合一绘画电脑类似，带有按钮的外接数位板笔有可能模拟所有三个鼠标点击。我见过。在我个人尝试的三款数位板中，推荐的是 GAOMON S620。它是专门为 9front 购买的，我最终在 OpenBSD 中用它绘制了大部分早期的 Techno-Mage。它在 9front 中仍然运行良好。即插即用。在 OpenBSD 中就不太行了。另外两款：表现得像触摸板（相对定位）Wacom CTL-672；无法工作 Wacom CTL-490。关于数位板的更多信息：[fqa] 3.2.4 - Tablet Digitizers。

1.2.4 Keyboard You can't really draw with a keyboard (though you can write pictures), but you can't get away without using it either. For tablet computers, inaccessible physical keyboards, and other obscure setups, there is bitsy/keyboard, an on-screen keyboard. It has to be started through rio with the -k option: % rio -k 'bitsy/keyboard -n'
1.2.4 键盘。你不能真正用键盘绘画（尽管你可以写出图片），但你也离不开它。对于平板电脑、无法访问的物理键盘和其他晦涩的设置，有一个屏幕键盘 bitsy/keyboard。它必须通过带有 -k 选项的 rio 启动：% rio -k 'bitsy/keyboard -n'

1.2.5 Drawterm / vnc Drawterm into a distant and/or virtual 9front computer, use any of the aforementioned methods and devices, and a million more. You'll probably have a bad time, but the size of your tool box will grow drastically. You could finger paint with 9front from your Android telephone, mouse draw from any Linux and MacOS, use all the input devices that don't work with 9front from Windows,... Any combination of these will impress your hypothetical employer. drawterm.9front.org
1.2.5 Drawterm / vnc。通过 Drawterm 连接到远程和/或虚拟的 9front 计算机，使用上述任何方法和设备，以及其他一百万种方法。你可能会有糟糕的体验，但你的工具箱规模将大幅增长。你可以通过安卓手机在 9front 上进行手指绘画，从任何 Linux 和 MacOS 上进行鼠标绘画，使用所有在 9front 中无法工作的 Windows 输入设备……这些组合中的任何一种都会给你的假想雇主留下深刻印象。drawterm.9front.org

1.3 The Medium All image operations, drawing, cropping,... are performed on files in 9front's internal image(6) format. This is the default for everything this document works with. For producing pictures this is irrelevant and you never have to think about it. It is only after you want to share your creation with the wider world that you will want to convert it to a different format. I usually name my 9front picture files file.pic. You can call them whatever you want.
1.3 媒介。所有的图像操作、绘画、裁剪等，都是在 9front 内部 image(6) 格式的文件上执行的。这是本文档所涉及的所有内容的默认格式。对于制作图片来说，这无关紧要，你永远不需要考虑它。只有当你想要与更广阔的世界分享你的创作时，你才需要将其转换为其他格式。我通常将我的 9front 图片文件命名为 file.pic。你可以随意命名。
