# STAGformer: A Spatio-temporal Agent Graph Transformer for Micro Mobility Demand Forecasting
# STAGformer：用于微出行需求预测的时空智能体图 Transformer

**Abstract:** Accurate station-level demand forecasting is essential for the efficient operation of bike-sharing systems, yet it remains challenging due to complex spatio-temporal dependencies and the large scale of urban networks. This paper presents STAGformer, a Spatio-Temporal Agent Graph Transformer that achieves efficient global modeling with linear computational complexity.

**摘要：** 精确的站点级需求预测对于共享单车系统的高效运营至关重要，但由于复杂的时空依赖性和城市网络的大规模特性，这仍然是一项挑战。本文提出了 STAGformer（时空智能体图 Transformer），它通过线性计算复杂度实现了高效的全局建模。

The model introduces a two-step agent attention mechanism, where a small set of learnable spatial and temporal agent tokens first aggregate global information and then broadcast it back to individual stations and time steps, effectively capturing long-range interactions while reducing the quadratic cost of standard self-attention to O(NT).

该模型引入了一种两步智能体注意力机制，其中一小组可学习的空间和时间智能体标记（agent tokens）首先聚合全局信息，然后将其广播回各个站点和时间步，从而在有效捕捉长距离交互的同时，将标准自注意力的二次方成本降低至 O(NT)。

STAGformer integrates four core modules: a spatio-temporal encoder that fuses dynamic node features with external contextual factors (weather, time, points of interest), a graph propagation module for spatial neighbor aggregation, a temporal convolution module for local pattern extraction, and the agent attention module for global dependency modeling.

STAGformer 集成了四个核心模块：一个将动态节点特征与外部环境因素（天气、时间、兴趣点）相融合的时空编码器、一个用于空间邻居聚合的图传播模块、一个用于局部模式提取的时间卷积模块，以及用于全局依赖建模的智能体注意力模块。

Extensive experiments on two real-world datasets -- NYC Citi-Bike and Chicago Divvy-Bike -- demonstrate that STAGformer consistently outperforms state-of-the-art baselines across multiple prediction horizons, achieving significant improvements in both RMSE and MAE. Ablation studies validate the contribution of each component, with the agent attention mechanism proving critical for modeling global spatio-temporal dependencies.

在两个真实世界数据集（纽约 Citi-Bike 和芝加哥 Divvy-Bike）上的大量实验表明，STAGformer 在多个预测时段内始终优于最先进的基准模型，并在 RMSE 和 MAE 指标上均取得了显著提升。消融实验验证了每个组件的贡献，其中智能体注意力机制被证明对于建模全局时空依赖关系至关重要。
