# $\chi$SAO: A GPU-Native Parallel Optimizer for Multimodal Black-Box Functions via Convergence-Anticonvergence Oscillation

**$\chi$SAO：一种基于收敛-反收敛振荡的多模态黑盒函数 GPU 原生并行优化器**

Finding all modes of a multimodal black-box function is a fundamental challenge in optimization, Bayesian inference, and scientific computing. Existing approaches -- basin-hopping, CMA-ES, multistart gradient descent -- operate sequentially and cannot exploit the massive parallelism of modern GPU hardware.

寻找多模态黑盒函数的所有极值点是优化、贝叶斯推断和科学计算中的一个基本挑战。现有的方法（如盆地跳跃法、CMA-ES、多起点梯度下降法）均以串行方式运行，无法利用现代 GPU 硬件的大规模并行能力。

We introduce $\chi$SAO (**C**onvergence-**H**alt-**I**nvert-**S**tick-**A**nd-**O**scillate), a GPU-native population optimizer that runs an entire sample batch simultaneously and exploits a deliberate convergence-anticonvergence oscillation cycle to escape local traps while freezing confirmed modes. The structural move is asymmetric: samples that reach true peaks are frozen (``stuck'') and preserved, while the rest keep exploring via momentum-based anti-convergence and stochastically smoothed gradients.

我们引入了 $\chi$SAO（收敛-停止-反转-锁定-振荡），这是一种 GPU 原生的种群优化器。它能够同时运行整个样本批次，并利用刻意的“收敛-反收敛”振荡周期来逃离局部陷阱，同时锁定已确认的极值点。其结构性移动是不对称的：到达真实峰值的样本会被冻结（“锁定”）并保留，而其余样本则通过基于动量的反收敛和随机平滑梯度继续探索。

Adaptive reseeding via two complementary strategies (Repulse Monkey and Golden Rooster) maintains population diversity throughout. On all 42 functions of the Simon Fraser University optimization benchmark suite across dimensions $d \in \{2, 4, 8, 16, 32, 64\}$, $\chi$SAO achieves **100%** mode recovery where all CPU baselines collapse at $d \geq 8$ on the hardest multimodal functions, at up to **$34\times$** speedup over basin-hopping on functions where all methods succeed (Michalewicz $d=64$) and up to **$39\times$** on unimodal functions (Rotated Hyper-Ellipsoid $d=64$, pure GPU dividend).

通过两种互补策略（Repulse Monkey 和 Golden Rooster）进行的自适应重采样，在整个过程中保持了种群的多样性。在西蒙弗雷泽大学（SFU）优化基准测试套件的所有 42 个函数（维度 $d \in \{2, 4, 8, 16, 32, 64\}$）上，$\chi$SAO 实现了 **100%** 的极值点恢复率；而在最困难的多模态函数上，当 $d \geq 8$ 时，所有 CPU 基准方法均失效。在所有方法都能成功的函数上（如 Michalewicz $d=64$），其速度比盆地跳跃法快达 **34 倍**；在单峰函数上（如旋转超椭球 $d=64$），得益于纯 GPU 加速，速度提升高达 **39 倍**。

All benchmarks evaluate the objective by value alone -- gradients come from finite differences -- so the reported speedups are a derivative-free worst case. Under substantial likelihood noise ($\sigma_{\mathrm{noise}}$ up to 1.0), mode detection remains 100% reliable. The algorithm is available as a standalone open-source Python package on PyPI.

所有基准测试仅通过函数值评估目标（梯度来自有限差分），因此所报告的加速比是在无导数情况下的最保守估计。在存在显著似然噪声（$\sigma_{\mathrm{noise}}$ 高达 1.0）的情况下，极值点检测依然保持 100% 的可靠性。该算法现已作为独立的开源 Python 包在 PyPI 上发布。
