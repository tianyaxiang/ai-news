# A multi-task spatiotemporal deep neural network for predicting penetration depth and morphology in laser welding
# 用于预测激光焊接熔深与形貌的多任务时空深度神经网络

In laser penetration welding, the assessment of penetration state and weld seam morphology plays a crucial role in determining the weld quality. This paper presents a comprehensive introduction of the innovative multi-task deep learning model that has the capability to predict penetration state, depth, and weld seam morphology with high accuracy.
在激光熔透焊中，对熔透状态和焊缝形貌的评估对于确定焊接质量至关重要。本文全面介绍了一种创新的多任务深度学习模型，该模型能够高精度地预测熔透状态、熔深以及焊缝形貌。

The monitoring platform relies on weld pool images captured during the laser welding process using a complementary metal-oxide-semiconductor (CMOS) camera. The proposed model integrates spatiotemporal features extracted from top weld pool images along with welding parameters, establishing a deep learning framework based on convolutional neural networks and state space models for more efficient extraction and processing of spatial-temporal information.
该监测平台依赖于在激光焊接过程中使用互补金属氧化物半导体（CMOS）相机拍摄的熔池图像。所提出的模型将从熔池顶部图像中提取的时空特征与焊接参数相结合，建立了一个基于卷积神经网络和状态空间模型的深度学习框架，从而更高效地提取和处理时空信息。

Furthermore, a reliable method for constructing the dataset is proposed to enhance both robustness and generalization capability of the developed model. Validation results on the test set demonstrate that prediction accuracy for penetration state can reach 99.35%, while prediction error for penetration depth is 1.79 millimeters, and accuracy of reconstructing the weld cross-section is 95.65%.
此外，本文提出了一种可靠的数据集构建方法，以增强所开发模型的鲁棒性和泛化能力。在测试集上的验证结果表明，熔透状态的预测准确率可达 99.35%，熔深预测误差为 1.79 毫米，焊缝横截面重构的准确率为 95.65%。

This study provides new insights and methodologies for in-situ quality control strategies in laser penetration welding systems.
本研究为激光熔透焊系统中的原位质量控制策略提供了新的见解和方法。
