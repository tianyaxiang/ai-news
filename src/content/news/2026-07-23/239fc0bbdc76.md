# LikeC4 / likec4

**Architecture as a code**
**代码即架构**

Visualize, collaborate on, and evolve your software architecture with always up-to-date, live diagrams generated from your code.
通过从代码生成的实时、始终保持更新的图表，可视化、协作并演进您的软件架构。

docs - playground - demo
文档 - 在线演练场 - 演示

**What is LikeC4?**
**什么是 LikeC4？**

Why "like"? LikeC4 is a modeling language for describing software architecture and tools to generate diagrams from the model. LikeC4 is inspired by C4 Model and Structurizr DSL, but provides some flexibility. You customize or define your own notation, element types, and any number of nested levels in architecture model. Perfectly tailored to your needs.
为什么叫 "like"？LikeC4 是一种用于描述软件架构的建模语言，以及从该模型生成图表的工具。LikeC4 受 C4 模型和 Structurizr DSL 的启发，但提供了更高的灵活性。您可以自定义或定义自己的符号、元素类型以及架构模型中任意数量的嵌套层级，完美契合您的需求。
