Apollo-11 🌐（支持多种语言）

Original Apollo 11 guidance computer (AGC) source code for Command Module (Comanche055) and Lunar Module (Luminary099). Digitized by the folks at Virtual AGC and MIT Museum. The goal is to be a repo for the original Apollo 11 source code. As such, PRs are welcome for any issues identified between the transcriptions in this repository and the original source scans for Luminary 099 and Comanche 055, as well as any files I may have missed.

这是阿波罗 11 号指令舱（Comanche055）和登月舱（Luminary099）原始阿波罗制导计算机（AGC）的源代码。由 Virtual AGC 和麻省理工学院博物馆（MIT Museum）的工作人员进行数字化处理。本项目旨在成为阿波罗 11 号原始源代码的存储库。因此，如果您发现本仓库中的转录内容与 Luminary 099 和 Comanche 055 的原始扫描件之间存在差异，或者发现我遗漏了任何文件，欢迎提交 PR（Pull Request）。

Comanche055: Part of the source code for Colossus 2A, the Command Module's (CM) Apollo Guidance Computer (AGC) for Apollo 11. Assemble revision 055 of AGC program Comanche by NASA2021113-051. 10:28 APR. 1, 1969.

Comanche055：Colossus 2A 源代码的一部分，这是阿波罗 11 号指令舱（CM）的阿波罗制导计算机（AGC）程序。由 NASA2021113-051 汇编的 AGC 程序 Comanche 的 055 修订版。1969 年 4 月 1 日 10:28。

Luminary099: Part of the source code for Luminary 1A, the Lunar Module's (LM) Apollo Guidance Computer (AGC) for Apollo 11. Assemble revision 001 of AGC program LMY99 by NASA2021112-061. 16:27 JUL. 14, 1969.

Luminary099：Luminary 1A 源代码的一部分，这是阿波罗 11 号登月舱（LM）的阿波罗制导计算机（AGC）程序。由 NASA2021112-061 汇编的 AGC 程序 LMY99 的 001 修订版。1969 年 7 月 14 日 16:27。
