# iOS code could reportedly let Apple cut off apps when users miss iPhone payments
# 报道称 iOS 代码或允许苹果在用户逾期未付 iPhone 款项时禁用应用

Code found in an iOS 27 beta would allow Apple to put a financed iPhone in “Restricted Mode” if it detects any missed payments, 9to5Mac reports. The finding follows a story from Bloomberg earlier this week claiming Apple is preparing to launch a new “Apple Upgrade” financing program for leasing new devices.

据 9to5Mac 报道，在 iOS 27 测试版中发现的代码显示，如果苹果检测到分期付款逾期，可以将该 iPhone 置于“受限模式”（Restricted Mode）。此前彭博社在本周早些时候曾报道称，苹果正准备推出一项名为“Apple Upgrade”的全新设备租赁融资计划。

iPhones in “Restricted Mode” would reportedly lose access to nearly all apps, except for basics like the Phone app, Settings, and the App Store. It uses a new “App Managed Features” system that allows financing or partner apps to continually check if a financed device is in good standing.

据报道，处于“受限模式”下的 iPhone 将无法访问几乎所有应用程序，仅保留电话、设置和 App Store 等基础功能。该模式使用了一套全新的“应用管理功能”（App Managed Features）系统，允许融资方或合作伙伴的应用持续检查分期设备的状态是否正常。

According to 9to5Mac, the code also includes a new “Partner Finance Lock” intended to block users from “erasing, reselling, or stripping a restricted device for parts.” While the feature is part of Find My, it reportedly doesn’t give financing partners access to the phone’s location.

据 9to5Mac 称，该代码还包含一项新的“合作伙伴金融锁”（Partner Finance Lock），旨在阻止用户“抹除、转售或拆解受限设备以获取零件”。虽然该功能属于“查找”（Find My）服务的一部分，但据称它不会向融资合作伙伴提供手机的地理位置信息。

This is from a beta version of iOS 27, so it’s possible this code and the new “Restricted Mode” could get changed or removed before Apple officially releases iOS 27. The “Apple Upgrade” program hasn’t been officially announced yet, either. Apple did not immediately respond to a request for comment.

由于这些信息来自 iOS 27 的测试版本，因此在苹果正式发布 iOS 27 之前，这些代码和新的“受限模式”仍有可能被更改或移除。“Apple Upgrade”计划也尚未正式公布。苹果公司未立即回应置评请求。
