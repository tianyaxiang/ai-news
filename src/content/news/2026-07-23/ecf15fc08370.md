# Style over Substance: A Shortcut Audit of Emotion-Description Preference Evaluation
# 重形式轻内容：对情感描述偏好评估的捷径审计

**Abstract:** Preference over model-generated emotion descriptions is emerging as a standard evaluation metric for multimodal emotion understanding, exemplified by the MER2026 MER-Prefer track on EmoPrefer. Such benchmarks assume that predicting the preferred description requires grounded cross-modal understanding of the video.

**摘要：** 对模型生成的情感描述进行偏好评估，正逐渐成为多模态情感理解的标准评估指标，以 EmoPrefer 上的 MER2026 MER-Prefer 赛道为例。此类基准测试假设预测偏好描述需要对视频进行扎实的跨模态理解。

We conduct a systematic shortcut audit of EmoPrefer using content-blind probes. A simple logistic regression using only description length and generator identity, without processing the text, video, or audio, performs comparably to LoRA-finetuned 7B text and audio-visual judges (65.8 versus 66.8 WAF on EmoPrefer-V2).

我们使用“内容盲测”（content-blind probes）对 EmoPrefer 进行了系统的捷径审计。仅使用描述长度和生成器身份进行简单的逻辑回归，在不处理文本、视频或音频的情况下，其表现与经过 LoRA 微调的 7B 文本和视听判别模型相当（在 EmoPrefer-V2 上分别为 65.8 对 66.8 WAF）。

Generator identity is recoverable from description text with 99.5 percent accuracy, every candidate pair contrasts two distinct generators, and the human preference labels agree with a fold-exclusive per-generator win-rate prior on 66 percent of the evaluated pairs. When the human label conflicts with this prior, trained judges still follow the style prior on 63 to 80 percent of the pairs.

从描述文本中恢复生成器身份的准确率高达 99.5%，每个候选对都对比了两个不同的生成器，且人类偏好标签在 66% 的评估对中与“基于生成器的胜率先验”一致。当人类标签与该先验冲突时，训练有素的判别模型在 63% 到 80% 的情况下仍会遵循这种风格先验。

On a length-matched subset that neutralizes verbosity bias, the tested media configurations yield no statistically significant improvement, while an ODIN-inspired diagnostic that decouples the style shortcut leaves its content head near chance. These results do not imply that human preferences are inherently stylistic or that the descriptions contain no emotional information. Instead, they show that the current scores can be reached without verifying either description against the video.

在消除冗长偏差的长度匹配子集上，所测试的媒体配置没有产生统计学上的显著改进；而一种受 ODIN 启发的、解耦了风格捷径的诊断方法，使其内容判断头的准确率接近随机水平。这些结果并不意味着人类偏好本质上就是风格化的，也不意味着描述中不包含情感信息。相反，它们表明当前的评分无需对照视频验证任何描述即可达成。

We recommend source-balanced pairing, strict length control, counter-stereotypical sliced reporting, and multi-annotator consensus for future cross-generator evaluations. Code is available at this https URL.

我们建议在未来的跨生成器评估中采用源平衡配对、严格的长度控制、反刻板印象的切片报告以及多标注者共识机制。代码已在链接中提供。
