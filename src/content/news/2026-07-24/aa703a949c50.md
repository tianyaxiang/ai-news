### Caw

**Launching today: Open source web terminal multiplexer for AI agents**
今天发布：面向 AI 智能体的开源 Web 终端多路复用器

**Caw is a web terminal multiplexer for OpenCode, Claude Code, Codex, Copilot, Antigravity, and Pi. It makes it easy to monitor and interact with multiple agent sessions from anywhere, especially on mobile.**
Caw 是一款专为 OpenCode、Claude Code、Codex、Copilot、Antigravity 和 Pi 设计的 Web 终端多路复用器。它让用户能够随时随地（尤其是在移动设备上）轻松监控并与多个智能体会话进行交互。

**Besides the terminal multiplexer itself, it includes:**
除了终端多路复用功能外，它还包含：

*   **Built-in file browser and editor**
    内置文件浏览器和编辑器
*   **Kanban-like board showing all running agents**
    类似看板的界面，用于展示所有正在运行的智能体
*   **Push notifications whenever an agent finishes working or requests your input**
    当智能体完成工作或需要你输入时，会发送推送通知
*   **Git worktrees on demand to parallelize work**
    按需使用 Git 工作树（worktrees）以实现并行工作
*   **Voice mode**
    语音模式
