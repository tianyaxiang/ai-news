# Croc GUI: 无需终端的加密点对点文件传输工具（跨平台）

**TL;DR:** Croc GUI 是一款为 `schollz/croc` 开发的免费桌面应用程序，支持加密的点对点文件传输，具备拖拽上传、二维码分享（通过 getcroc.com）以及局域网模式。支持 macOS、Windows 和 Linux，采用 MIT 协议开源。下载地址：[GitHub Releases](https://github.com/interfluve-wav/croc-gui/releases)。

**Why I built this**
I send files with croc constantly. End-to-end encrypted, cross-platform, no vendor cloud. The CLI is perfect — until you're helping someone who doesn't have a terminal open. Croc GUI is the Send/Receive desktop app I wanted: same croc binary, clearer UX.

**开发初衷**
我经常使用 croc 发送文件。它具备端到端加密、跨平台且不依赖任何厂商云服务等优点。命令行界面（CLI）非常完美，但当你需要帮助那些不熟悉终端操作的人时，它就显得有些不便。Croc GUI 正是我想要的发送/接收桌面应用：它使用相同的 croc 二进制文件，但提供了更直观的用户体验。

**What it does**
*   **Send** — drag files/folders, get a code phrase + QR link
*   **Receive** — paste a code, pick a download folder
*   **Share** — copy phrase, getcroc.com URL, or full croc … command
*   **Local-only** — croc --local for LAN peers
*   **Zip** — pack on send, unpack helper on receive
*   **Options** — relay, port, proxy, overwrite, auto-confirm

**功能特性**
*   **发送** — 拖拽文件/文件夹，获取代码短语 + 二维码链接。
*   **接收** — 粘贴代码，选择下载文件夹。
*   **分享** — 复制短语、getcroc.com URL 或完整的 croc 命令。
*   **仅限本地** — 使用 `croc --local` 进行局域网传输。
*   **压缩** — 发送时自动打包，接收时自动解压。
*   **选项** — 支持中继（relay）、端口、代理、覆盖文件、自动确认等设置。

**What it doesn't do**
*   Reimplement croc's crypto or protocol
*   Upload anything to a GUI-specific cloud
*   Claim to be an official schollz project

**它不做什么**
*   重新实现 croc 的加密算法或协议。
*   将任何数据上传到 GUI 专属的云端。
*   声称是 schollz 的官方项目。

**Stack**
*   UI: React + TypeScript
*   Shell: Tauri 2 (Rust)
*   Engine: bundled croc binary per platform

**技术栈**
*   UI：React + TypeScript
*   外壳：Tauri 2 (Rust)
*   引擎：各平台内置的 croc 二进制文件

**Dev quick start**
```bash
git clone https://github.com/interfluve-wav/croc-gui.git
cd croc-gui/gui
npm install
npm run bundle:croc:download
npm run tauri:dev
```
