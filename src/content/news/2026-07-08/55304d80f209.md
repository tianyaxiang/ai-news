# Federated Learning for Object Detection: Enabling Collaborative Drone Learning Without Centralizing Data
# 面向目标检测的联邦学习：在无需数据中心化的情况下实现无人机协同学习

**Abstract:** Object detection is a fundamental capability for AI-driven perception in safety-critical drone and edge-vision systems, including disaster response, operational security environments, infrastructure monitoring and defense applications. Robust model performance in such environments depends on large, continuously updated datasets. However, training high-performing detectors typically requires centralizing aerial imagery, which raises privacy, regulatory, storage, and bandwidth challenges. This is especially problematic in distributed drone deployments, where visual data is generated onboard and is often impractical or undesirable to transfer to a centralized infrastructure.

**摘要：** 目标检测是人工智能驱动的感知系统在安全关键型无人机和边缘视觉系统中的核心能力，涵盖了灾难响应、运营安全环境、基础设施监控及国防应用等领域。在这些环境中，稳健的模型性能依赖于大规模且持续更新的数据集。然而，训练高性能检测器通常需要将航空影像进行中心化处理，这带来了隐私、监管、存储和带宽方面的挑战。在分布式无人机部署中，这一问题尤为突出，因为视觉数据是在机载端生成的，将其传输到中心化基础设施往往既不切实际也不理想。

In this work, we apply Federated Learning (FL) for object detection, enabling drones to improve a shared model while keeping image data local and private. We implement a federated object detection pipeline using the FL platform on the KIIT-MiTA dataset, and compare it with Single-drone and Centralized baselines using mean Average Precision (mAP) at IoU thresholds of 0.50 and 0.50-0.95.

在这项工作中，我们将联邦学习（FL）应用于目标检测，使无人机能够在保持图像数据本地化和私密性的同时，共同改进共享模型。我们利用联邦学习平台在 KIIT-MiTA 数据集上实现了一个联邦目标检测流水线，并将其与单机训练（Single-drone）和中心化训练（Centralized）基准进行了对比，评估指标为 IoU 阈值在 0.50 和 0.50-0.95 下的平均精度均值（mAP）。

In our experiments, the proposed FL approach remains close to Centralized training while dramatically improving over Single-drone training, with the best lightweight model (YOLO26 nano), suitable for deployment even on very limited edge infrastructure, achieving relative gains of 52.89% and 67.80% in mAP@0.50 and mAP@0.50:0.95, respectively. These results show that FL enables scalable, high-performing, and privacy-preserving object detection across distributed drone fleets without data centralization.

实验结果表明，所提出的联邦学习方法在性能上接近中心化训练，且显著优于单机训练。其中，最轻量级的模型（YOLO26 nano）即使在资源极其有限的边缘基础设施上也能部署，其 mAP@0.50 和 mAP@0.50:0.95 的相对增益分别达到了 52.89% 和 67.80%。这些结果证明，联邦学习能够在无需数据中心化的情况下，实现分布式无人机群中可扩展、高性能且保护隐私的目标检测。
