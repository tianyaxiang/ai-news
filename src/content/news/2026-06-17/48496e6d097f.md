# AI for Maritime Security: Comparative Evaluation of CNN and Vision Transformer Architectures for Maritime Object Detection
# 海上安全人工智能：CNN 与 Vision Transformer 架构在海上目标检测中的对比评估

**Abstract:** This study aims to enhance maritime security by using advanced Artificial Intelligence (AI) and Computer Vision (CV) techniques. For this purpose, it was designed and assessed intelligent object detection systems that can detect the presence of ships on the sea surface under different real-time environments.

**摘要：** 本研究旨在利用先进的人工智能 (AI) 和计算机视觉 (CV) 技术来加强海上安全。为此，研究设计并评估了智能目标检测系统，该系统能够在不同的实时环境下检测海面船舶的存在。

To achieve this goal, a maritime image dataset with 6,468 images was used, covering different weather conditions like cloudy, foggy, rainy, and sunny environments. Six deep learning architectures were evaluated, including a base Convolutional Neural Network (CNN) model, four transfer learning models (Xception, VGG16, MobileNetV2, and EfficientNetV2L), and a Vision Transformer (ViT) model.

为实现这一目标，研究使用了一个包含 6,468 张图像的海上图像数据集，涵盖了多云、多雾、多雨和晴天等不同天气条件。研究评估了六种深度学习架构，包括一个基础卷积神经网络 (CNN) 模型、四个迁移学习模型（Xception、VGG16、MobileNetV2 和 EfficientNetV2L）以及一个视觉 Transformer (ViT) 模型。

The models were compared using multiple performance indicators, including accuracy, Type I and Type II errors, model size, and video processing time. The results show that model performance varies depending on computational constraints and deployment conditions.

研究通过多个性能指标对这些模型进行了比较，包括准确率、第一类和第二类错误率、模型大小以及视频处理时间。结果表明，模型性能会根据计算约束和部署条件的不同而有所差异。

While lightweight architectures are suitable for resource-limited devices, the ViT achieved the best overall performance, reaching 100% accuracy with the lowest error rates and the fastest video processing time. The findings highlight the potential of AI-driven computer vision systems for maritime surveillance, border protection, and autonomous navigation.

虽然轻量级架构适用于资源受限的设备，但 ViT 实现了最佳的整体性能，达到了 100% 的准确率，且错误率最低、视频处理速度最快。这些研究结果凸显了人工智能驱动的计算机视觉系统在海上监视、边境保护和自主导航方面的潜力。
