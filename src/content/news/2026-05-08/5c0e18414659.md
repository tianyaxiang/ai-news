# MP-ISMoE: Mixed-Precision Interactive Side Mixture-of-Experts for Efficient Transfer Learning
# MP-ISMoE：用于高效迁移学习的混合精度交互式侧边专家混合模型

**Abstract:** Parameter-efficient transfer learning (PETL) has emerged as a pivotal paradigm for adapting pre-trained foundation models to downstream tasks, significantly reducing trainable parameters yet suffering from substantial memory overhead caused by gradient backpropagation during fine-tuning. 

**摘要：** 参数高效迁移学习（PETL）已成为将预训练基础模型适配到下游任务的关键范式。它虽然显著减少了可训练参数，但在微调过程中的梯度反向传播却带来了巨大的内存开销。

While memory-efficient transfer learning (METL) circumvents this challenge by bypassing backbone gradient computation via lightweight small side networks, its stringent memory constraint severely limits learning capacity of side networks, thereby significantly compromising performance. 

尽管内存高效迁移学习（METL）通过轻量级侧边网络绕过主干网络的梯度计算来规避这一挑战，但其严格的内存限制严重制约了侧边网络的学习能力，从而显著损害了模型性能。

To address these limitations, we propose a novel Mixed-Precision Interactive Side Mixture-of-Experts framework (MP-ISMoE). Specifically, we first propose a Gaussian Noise Perturbed Iterative Quantization (GNP-IQ) scheme to quantize weights into lower-bits while effectively decreasing quantization errors. 

为了解决这些局限性，我们提出了一种新颖的混合精度交互式侧边专家混合框架（MP-ISMoE）。具体而言，我们首先提出了一种高斯噪声扰动迭代量化（GNP-IQ）方案，在将权重压缩为低比特的同时，有效降低了量化误差。

By leveraging memory conserved from GNP-IQ, we subsequently employ Interactive Side Mixture-of-Experts (ISMoE) to scaling up side networks without sacrificing overall memory efficiency. 

通过利用 GNP-IQ 节省出的内存，我们随后采用了交互式侧边专家混合（ISMoE）技术，在不牺牲整体内存效率的前提下扩展了侧边网络的规模。

Different from conventional mixture-of-experts, ISMoE learns to select optimal experts by interacting with salient features from frozen backbones, thus suppressing knowledge forgetting and boosting performance. 

与传统的专家混合模型不同，ISMoE 通过与冻结主干网络中的显著特征进行交互来学习选择最优专家，从而抑制了知识遗忘并提升了性能。

Extensive experiments across diverse vision-language and language-only tasks demonstrate that MP-ISMoE remarkably promotes accuracy compared to state-of-the-art METL approaches, while maintaining comparable parameter and memory efficiency.

在多种视觉-语言和纯语言任务上的广泛实验表明，与最先进的 METL 方法相比，MP-ISMoE 在保持相当的参数和内存效率的同时，显著提升了准确率。
