### ClearMesh

**ClearMesh**

ClearMesh brings Git-like version control to large files. It helps AI, VFX, research, data, and engineering teams commit, push, clone, sync, branch, and mount datasets, model files, media assets, CAD exports, and binary folders.
ClearMesh 为大文件带来了类似 Git 的版本控制功能。它旨在帮助人工智能、视觉特效（VFX）、科研、数据及工程团队对数据集、模型文件、媒体资源、CAD 导出文件和二进制文件夹进行提交（commit）、推送（push）、克隆（clone）、同步（sync）、分支（branch）和挂载（mount）操作。

Files are stored as chunks in S3/R2-compatible Vault storage, with optional client-side encryption.
文件以数据块（chunks）的形式存储在兼容 S3/R2 的 Vault 存储中，并支持可选的客户端加密。

Unchanged chunks can be reused across versions, and repos can be mounted read-only so tools can stream files from a normal path.
未更改的数据块可以在不同版本间重复使用；仓库还可以以只读方式挂载，以便各类工具能够通过常规路径直接读取文件流。
