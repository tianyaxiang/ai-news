# Anthropic acquires Stainless
# Anthropic 收购 Stainless

May 18, 2026
2026年5月18日

The frontier of AI is shifting from models that answer to agents that act—and agents are only as capable as the systems they can reach. Today, Anthropic is acquiring Stainless, a leader in SDKs and MCP server tooling, to extend that reach even further.
人工智能的前沿正从“回答问题的模型”转向“采取行动的智能体”——而智能体的能力取决于它们所能触达的系统。今天，Anthropic 宣布收购 SDK 和 MCP 服务器工具领域的领导者 Stainless，以进一步扩展这种触达能力。

Founded in 2022, Stainless has powered the generation of every official Anthropic SDK since the earliest days of our API. Hundreds of companies rely on Stainless to generate SDKs, CLIs, and MCP servers—the libraries, command-line tools, and connectors that let developers and agents use an API. Stainless turns an API spec into SDKs across TypeScript, Python, Go, Java, Kotlin, and more. Each one is fast, reliable, and built to feel native in its language.
Stainless 成立于 2022 年，自我们 API 推出初期起，就一直为 Anthropic 的所有官方 SDK 提供生成支持。数百家公司依赖 Stainless 来生成 SDK、CLI 和 MCP 服务器——即让开发者和智能体能够使用 API 的库、命令行工具和连接器。Stainless 能将 API 规范转化为支持 TypeScript、Python、Go、Java、Kotlin 等多种语言的 SDK。每一个生成的 SDK 都快速、可靠，且在使用体验上与原生语言无异。

“Stainless has shaped how developers experience the Claude API since the start, and it’s been great to work with them on that,” said Katelyn Lesse, Head of Platform Engineering at Anthropic. “Agents are only as useful as what they can connect to. We’re excited to bring the Stainless team into Anthropic to advance Claude’s ability to connect to data and tools.”
“自 Claude API 诞生以来，Stainless 一直在塑造开发者的使用体验，与他们合作的过程非常愉快，”Anthropic 平台工程主管 Katelyn Lesse 表示，“智能体的价值取决于它们能连接什么。我们很高兴能将 Stainless 团队引入 Anthropic，以提升 Claude 连接数据和工具的能力。”

“I started Stainless because SDKs deserve as much care as the APIs they wrap. Anthropic was one of the first teams to bet on this with us,” said Alex Rattray, Founder and CEO of Stainless. “We have been watching what developers have built on Claude over the last few years, which made bringing our teams together an easy decision. The team gets to keep doing the work we love, on the platform where it matters most.”
“我创办 Stainless 是因为 SDK 应该像它们所封装的 API 一样受到重视。Anthropic 是最早与我们共同践行这一理念的团队之一，”Stainless 创始人兼首席执行官 Alex Rattray 说道，“过去几年，我们一直关注着开发者在 Claude 上构建的应用，这使得合并团队成为一个轻松的决定。我们的团队将能够在最重要的平台上，继续从事我们热爱的工作。”

Anthropic created MCP to make agent connectivity possible. By bringing together the Stainless and Anthropic teams, the Claude Platform continues to push the frontier of developer experience and agent connectivity.
Anthropic 创建了 MCP（模型上下文协议）以实现智能体连接。通过将 Stainless 和 Anthropic 团队整合在一起，Claude 平台将继续推动开发者体验和智能体连接的前沿发展。
