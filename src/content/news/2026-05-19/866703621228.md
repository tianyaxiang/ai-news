# Five years later, Windows 11 brings back much-missed taskbar options (and more)
# 五年之后，Windows 11 终于带回了备受期待的任务栏选项（及更多功能）

When Windows 11 launched in 2021, we mostly liked its refreshed look—the rounded corners and menus with just a hint of translucency were a nice change from the flat colors and hard corners of the Windows 8/10 era. But its reformulated taskbar and Start menu came with a number of functional regressions from the versions in Windows 10. Some of these were addressed quickly; others continue to linger.
当 Windows 11 在 2021 年发布时，我们大多对其焕然一新的外观表示赞赏——圆角设计和带有半透明感的菜单，相比 Windows 8/10 时代的扁平色彩和硬朗边角，确实是一种不错的改变。然而，其重构后的任务栏和“开始”菜单在功能上相比 Windows 10 版本出现了一些倒退。其中一些问题很快得到了解决，但另一些问题却一直存在。

A new Windows Insider Preview build released to testers includes a new wave of improvements that fix longstanding regressions while trying out new things. Most significantly, the Windows 11 taskbar can now be docked to any edge of your screen, including the left and right, something that was possible in Windows 10 (and many older versions of Windows) but has been missing from Windows 11 since launch.
微软向测试人员发布的一个新的 Windows Insider 预览版包含了一系列改进，在尝试新功能的同时修复了长期存在的倒退问题。最显著的变化是，Windows 11 任务栏现在可以停靠在屏幕的任何边缘，包括左侧和右侧。这在 Windows 10（以及许多更早的 Windows 版本）中是基本功能，但自 Windows 11 发布以来一直缺失。

Users can configure slightly different taskbar behavior for every taskbar position—if you prefer a different icon alignment or a left/right-mounted taskbar over a top/bottom-mounted taskbar, or if you want different settings for labels and icon groupings, you can choose different options for each position and Windows will remember them.
用户可以为每个任务栏位置配置略有不同的行为——如果你偏好不同的图标对齐方式，或者更喜欢左右停靠而非上下停靠的任务栏，亦或是希望对标签和图标分组进行不同的设置，你可以为每个位置选择不同的选项，Windows 会记住这些设置。

Microsoft says there are several features that haven’t been implemented yet—the taskbar won’t auto-hide in any of the alternate positions, and the “tablet-optimized taskbar” with larger, more finger-friendly icon sizes and spacing also isn’t supported. Touch gestures and the Search box also aren’t supported. All of these features are coming at some point; they just aren’t ready now. Microsoft is “evaluating additional features like different taskbar positions per monitor” for multi-monitor setups.
微软表示，仍有几项功能尚未实现——任务栏在任何非默认位置下都无法自动隐藏，且支持更大、更适合手指操作的图标尺寸和间距的“平板优化任务栏”也暂不支持。触摸手势和搜索框目前也不支持。所有这些功能未来都会推出，只是现在还没准备好。微软目前正在评估针对多显示器设置的“不同显示器可设置不同任务栏位置”等附加功能。

Another change Microsoft is testing makes the taskbar and all its icons smaller, a change designed to increase the amount of vertical space available on smaller screens without requiring users to fully hide the taskbar.
微软正在测试的另一项更改是缩小任务栏及其所有图标，此举旨在增加小屏幕上的垂直可用空间，而无需用户完全隐藏任务栏。

Microsoft is also making a handful of changes to the Start menu, including a user-selectable size setting (previously, the menu would increase and decrease in size dynamically based on the size of your display). Each section of the Start menu (the pinned apps section, the “recommended” section, and the “all apps” section) will be individually toggleable. And users will be allowed to hide “recommended” apps advertised from the Microsoft store while still being able to see jump lists and recent files in the File Explorer.
微软还对“开始”菜单进行了一些调整，包括用户可自定义大小的设置（此前，菜单会根据显示器尺寸动态调整大小）。“开始”菜单的每个部分（固定应用区域、“推荐”区域和“所有应用”区域）现在都可以单独开关。此外，用户将能够隐藏来自微软商店的“推荐”应用广告，同时仍能在文件资源管理器中查看跳转列表和最近使用的文件。

Users who decide to keep the “recommended” apps section visible should also benefit from “improv[ed] file relevancy” that “better reflect[s] what you have been working on.”
选择保留“推荐”应用区域的用户，也将受益于“改进的文件相关性”，该功能将“更好地反映你最近的工作内容”。

Some of these changes are available in the current Windows Insider Preview builds in the Experimental channel (which replaced the Canary and Dev channels in Microsoft’s latest beta program shake-up). Others will be released “over the coming weeks,” at which point more polished and refined versions will presumably come to the Beta channel and, eventually, the public version of Windows 11.
其中一些更改已在当前的 Windows Insider 预览版“实验频道”（Experimental channel，该频道取代了微软最新测试计划调整中的 Canary 和 Dev 频道）中提供。其他更改将在“未来几周内”发布，届时更完善的版本预计会进入 Beta 频道，并最终推送至 Windows 11 的正式版中。

All of these changes are being made as part of Microsoft’s continued “commitment to Windows quality” push, meant to address several of the real and perceived shortcomings of Windows 11 relative to older versions.
所有这些改进都是微软持续推动“致力于提升 Windows 质量”计划的一部分，旨在解决 Windows 11 相较于旧版本在实际体验和用户感知上存在的一些不足。
