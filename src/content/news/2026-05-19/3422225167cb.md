# AgentStop: Terminating Local AI Agents Early to Save Energy in Consumer Devices
# AgentStop：提前终止本地 AI 智能体以节省消费设备电量

Autonomous agents powered by large language models (LLMs) are increasingly used to automate complex, multi-step tasks such as coding or web-based question answering. While remote, cloud-based agents offer scalability and ease of deployment, they raise privacy concerns, depend on network connectivity, and incur recurring API costs.
由大语言模型（LLM）驱动的自主智能体正越来越多地被用于自动化处理复杂的、多步骤的任务，例如编程或基于网络的问答。虽然基于云端的远程智能体具有可扩展性和易于部署的优势，但它们引发了隐私担忧，依赖网络连接，并会产生持续的 API 使用成本。

Deploying agents locally on user devices mitigates these issues by preserving data privacy and eliminating usage-based fees. However, agentic workflows are far more resource-intensive than typical LLM interactions. Iterative reasoning, tool use, and failure retries substantially increase token consumption, often expending significant compute without successfully completing tasks.
在用户设备上本地部署智能体可以通过保护数据隐私和消除按量付费来缓解这些问题。然而，智能体工作流比典型的 LLM 交互更消耗资源。迭代推理、工具使用和失败重试会大幅增加 Token 消耗，往往在未能成功完成任务的情况下就消耗了大量的计算资源。

In this work, we investigate the time, token, and energy overhead of locally deployed LLM-based agents on consumer hardware. Our measurements show that agentic execution increases GPU power draw, temperature, and battery drain compared to single-inference workloads.
在这项工作中，我们研究了在消费级硬件上本地部署基于 LLM 的智能体所带来的时间、Token 和能源开销。我们的测量结果表明，与单次推理工作负载相比，智能体执行会增加 GPU 的功耗、温度和电池消耗。

To address this inefficiency, we introduce AgentStop, a lightweight efficiency supervisor that predicts and preemptively terminates trajectories unlikely to succeed. Leveraging low-cost execution signals, such as token-level log probabilities, AgentStop can reduce wasted energy by 15-20% with minimal impact on task performance (<5% utility drop) for challenging web-based question answering and coding benchmarks.
为了解决这种低效问题，我们引入了 AgentStop，这是一个轻量级的效率监督器，它能够预测并预先终止那些不太可能成功的任务路径。通过利用低成本的执行信号（如 Token 级别的对数概率），AgentStop 可以在极小影响任务性能（效用下降 <5%）的前提下，在具有挑战性的网页问答和编程基准测试中减少 15-20% 的能源浪费。

These findings position predictive early termination as a practical mechanism for enabling sustainable, privacy-preserving LLM agents on user devices. Our project code and data are available at this https URL.
这些发现表明，预测性提前终止是一种实用的机制，能够助力在用户设备上实现可持续且保护隐私的 LLM 智能体。我们的项目代码和数据可在该链接获取。
