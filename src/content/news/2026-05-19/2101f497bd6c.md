# Haiku OS runs on M1 Macs now
# Haiku OS 现已可在 M1 Mac 上运行

Big news from the Haiku forums: the Haiku ARM port is running on M1 Macs now.
来自 Haiku 论坛的重磅消息：Haiku 的 ARM 移植版现已可以在 M1 Mac 上运行。

This is bare metal, no VM. m1n1+u-boot deal with the Apple-specific parts of booting, so we can boot UEFI images from USB like any PC. ↫ smrobtzz on the Haiku forums
这是在裸机上运行，而非虚拟机。通过 m1n1 和 u-boot 处理苹果特有的引导部分，我们现在可以像在普通 PC 上一样，通过 USB 引导 UEFI 镜像。—— 引自 Haiku 论坛用户 smrobtzz

USB is apparently broken, but all 8 cores are functional, and it boots to a desktop. It’s still early days, for the ARM port in general and the M1 Mac port specifically, but it’s a great start.
虽然 USB 功能目前似乎还无法使用，但所有 8 个核心均已正常工作，并且可以成功进入桌面环境。无论是对于 ARM 移植版整体，还是针对 M1 Mac 的移植而言，这都还处于早期阶段，但这是一个非常棒的开始。
