# TailMux

**Launching today**
今天正式发布

**Multiple Tailscale tailnets at once, no switching + no VM**
同时连接多个 Tailscale 网络，无需切换，无需虚拟机

**VPN client**
VPN 客户端

**The official Tailscale client keeps one tailnet active at a time. TailMux makes work and personal tailnets reachable simultaneously on macOS and Linux, without switching accounts, running multiple system daemons, or using a VM.**
官方 Tailscale 客户端一次只能保持一个网络（tailnet）处于活跃状态。TailMux 让用户可以在 macOS 和 Linux 上同时访问工作和个人网络，无需切换账户、运行多个系统守护进程，也不需要使用虚拟机。

**It runs an isolated embedded node per profile and routes by hostname, with strict no-fallback isolation. Use SSH, RDP/SMB, browsers, curl, git, and npm across tailnets at the same time.**
它为每个配置文件运行一个隔离的嵌入式节点，并按主机名进行路由，具有严格的无回退隔离机制。你可以同时跨多个网络使用 SSH、RDP/SMB、浏览器、curl、git 和 npm。
