**What are Skills?**
Skills are markdown files that give AI agents specialized knowledge and workflows for specific tasks. When you add these to your project, your agent can recognize when you're working on a marketing task and apply the right frameworks and best practices.

**什么是技能（Skills）？**
技能是 Markdown 文件，为 AI 智能体提供针对特定任务的专业知识和工作流。当你将这些文件添加到项目中时，智能体能够识别你正在进行的营销任务，并应用相应的框架和最佳实践。

**How Skills Work Together**
Skills reference each other and build on shared context. The product-marketing skill is the foundation — every other skill checks it first to understand your product, audience, and positioning before doing anything.

**技能如何协同工作**
各项技能之间相互引用，并建立在共享的上下文之上。`product-marketing`（产品营销）技能是基础——其他所有技能在执行任何操作前，都会先读取该技能，以了解你的产品、受众和定位。

**Available Skills**

*   **ads**: When the user wants help with paid advertising campaigns on Google Ads, Meta (Facebook/Instagram), LinkedIn, Twitter/X.
*   **ai-seo**: When the user wants to optimize content for AI search engines, get cited by LLMs, or appear in AI-generated answers.
*   **analytics**: When the user wants to set up, improve, or audit analytics tracking and measurement.
*   **aso**: When the user wants to audit or optimize an App Store or Google Play listing.
*   **churn-prevention**: When the user wants to reduce churn, build cancellation flows, set up save offers, recover failed payments.
*   **co-marketing**: When the user wants to find co-marketing partners, plan joint campaigns, or brainstorm partnership opportunities.
*   **cold-email**: Write B2B cold emails and follow-up sequences that get replies.
*   **community-marketing**: Build and leverage online communities to drive product growth and brand loyalty.
*   **competitor-profiling**: When the user wants to research, profile, or analyze competitors from their URLs.
*   **competitors**: When the user wants to create competitor comparison or alternative pages for SEO and sales enablement.
*   **content-strategy**: When the user wants to plan a content strategy, decide what content to create, or figure out what topics to cover.
*   **copy-editing**: When the user wants to edit, review, or improve existing marketing copy, or refresh outdated content.
*   **copywriting**: When the user wants to write, rewrite, or improve marketing copy for any page — including homepage, landing pages.
*   **cro**: When the user wants to optimize, improve, or increase conversions on any marketing page or form.
*   **customer-research**: When the user wants to conduct, analyze, or synthesize customer research.
*   **directory-submissions**: When the user wants to submit their product to startup, SaaS, AI, agent, MCP, no-code, or review directories.
*   **emails**: When the user wants to create or optimize an email sequence, drip campaign, automated email flow, or lifecycle email.
*   **free-tools**: When the user wants to plan, evaluate, or build a free tool for marketing purposes.
*   **image**: When the user wants to create, generate, edit, or optimize images for marketing.
*   **launch**: When the user wants to plan a product launch, feature announcement, or release strategy.

*   **community-marketing（社区营销）**：构建并利用在线社区来推动产品增长和品牌忠诚度。

*   **competitors（竞争对手对比）**：当用户想要为 SEO 和销售赋能创建竞争对手对比页面或替代方案页面时使用。

*   **copy-editing（文案编辑）**：当用户想要编辑、审阅或改进现有营销文案，或更新过时内容时使用。
*   **copywriting（文案写作）**：当用户想要为任何页面（包括主页、落地页）撰写、重写或改进营销文案时使用。
*   **cro（转化率优化）**：当用户想要优化、改进或提高任何营销页面或表单的转化率时使用。
*   **customer-research（客户研究）**：当用户想要进行、分析或综合客户研究时使用。
*   **directory-submissions（目录提交）**：当用户想要将产品提交到初创公司、SaaS、AI、智能体、MCP、无代码或评论目录时使用。
*   **emails（邮件营销）**：当用户想要创建或优化邮件序列、滴灌营销、自动化邮件流或生命周期邮件时使用。
*   **free-tools（免费工具）**：当用户想要为营销目的（如潜在客户挖掘、SEO 价值）规划、评估或构建免费工具时使用。
*   **image（图像处理）**：当用户想要为营销目的创建、生成、编辑或优化图像时使用。
*   **launch（产品发布）**：当用户想要规划产品发布、功能公告或发布策略时使用。
