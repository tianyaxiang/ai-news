# Mini-Batch Class Composition Bias in Link Prediction
# 链路预测中的小批量类别组成偏差

**Abstract:** Prior work on node classification has shown that Graph Neural Networks (GNNs) can learn representations that transfer across graphs, when underlying graph properties are shared. For a fixed graph, one would then expect GNNs trained for link prediction to learn a representation consistent with that learnt for node classification.

**摘要：** 先前关于节点分类的研究表明，当底层图属性共享时，图神经网络（GNN）可以学习到在不同图之间迁移的表示。对于固定的图，人们通常会预期为链路预测训练的 GNN 所学习到的表示，应与节点分类任务中所学习到的表示保持一致。

We show this intuition does not hold in the general case. Instead, we find popular link prediction models can learn a trivial mini-batch dependent heuristic, enabled by batch-normalisation layers, to solve the edge classification task.

我们证明了这种直觉在一般情况下并不成立。相反，我们发现流行的链路预测模型可以通过批归一化（batch-normalisation）层，学习到一种依赖于小批量（mini-batch）的简单启发式方法来解决边分类任务。

When correcting for this, we observe increased alignment of the network representation with node-class relevant features, suggesting the network has learnt a graph representation that better aligns with the underlying graph's properties.

在对此进行修正后，我们观察到网络表示与节点类别相关特征之间的一致性有所提高，这表明网络已经学习到了一种能更好地与底层图属性相契合的图表示。

Our findings suggest that standard link prediction training may be leading us to overestimate link predictors' ability to learn a generalised representation of a graph that is consistent across tasks.

我们的研究结果表明，标准的链路预测训练可能导致我们高估了链路预测器学习跨任务一致的图通用表示的能力。
