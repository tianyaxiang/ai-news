### mold-desktop

**mold-desktop** is a programmable desktop environment built primarily using Common Lisp. It is an open-source project hosted on GitHub, designed to offer a highly customizable and extensible user experience.

**mold-desktop** 是一个主要使用 Common Lisp 构建的可编程桌面环境。这是一个托管在 GitHub 上的开源项目，旨在提供高度可定制和可扩展的用户体验。

***

The project repository contains a wide array of modules and core components, including support for command prompts, Kanban views, file system management, and various integrations like Google Search, GDrive, and IGDB. The codebase is actively maintained, with recent updates focusing on command handling, system housekeeping, and modular improvements.

该项目仓库包含大量的模块和核心组件，包括对命令提示符、看板视图、文件系统管理的支持，以及与 Google 搜索、GDrive 和 IGDB 等服务的各种集成。代码库目前处于活跃维护状态，最近的更新集中在命令处理、系统清理和模块化改进上。

***

**Technical Breakdown:**
*   **Primary Language:** Common Lisp (90.2%)
*   **Secondary Languages:** CSS (4.3%), JavaScript (3.9%), Java (0.9%), HTML (0.5%)
*   **Key Features:** Programmable interface, modular architecture, webview integration, and support for deployment via Docker to platforms like fly.io.

**技术概览：**
*   **主要语言：** Common Lisp (90.2%)
*   **次要语言：** CSS (4.3%), JavaScript (3.9%), Java (0.9%), HTML (0.5%)
*   **核心功能：** 可编程界面、模块化架构、Webview 集成，以及支持通过 Docker 部署到 fly.io 等平台。
