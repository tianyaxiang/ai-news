# Title: aaif-goose / goose

**🦆 goose has moved! This project has moved from block/goose to the Agentic AI Foundation (AAIF) at the Linux Foundation. Some links and references are still being updated — please bear with us during the transition.**

🦆 goose 搬家了！本项目已从 block/goose 迁移至 Linux 基金会旗下的智能体 AI 基金会 (AAIF)。部分链接和引用仍在更新中，过渡期间请您谅解。

**goose your native open source AI agent — desktop app, CLI, and API — for code, workflows, and everything in between. goose is a general-purpose AI agent that runs on your machine. Not just for code — use it for research, writing, automation, data analysis, or anything you need to get done.**

goose 是您的原生开源 AI 智能体——提供桌面应用、命令行工具 (CLI) 和 API——适用于代码编写、工作流处理以及介于两者之间的任何任务。goose 是一款运行在您本地机器上的通用 AI 智能体。它不仅限于代码编写，还可用于研究、写作、自动化、数据分析或任何您需要完成的工作。

**A native desktop app for macOS, Linux, and Windows. A full CLI for terminal workflows. An API to embed it anywhere. Built in Rust for performance and portability.**

它提供适用于 macOS、Linux 和 Windows 的原生桌面应用，支持终端工作流的完整 CLI，以及可嵌入任何位置的 API。采用 Rust 构建，兼顾性能与可移植性。

**goose works with 15+ providers — Anthropic, OpenAI, Google, Ollama, OpenRouter, Azure, Bedrock, and more. Use API keys or your existing Claude, ChatGPT, or Gemini subscriptions via ACP. Connect to 70+ extensions via the Model Context Protocol open standard.**

goose 支持 15 家以上的服务提供商，包括 Anthropic、OpenAI、Google、Ollama、OpenRouter、Azure、Bedrock 等。您可以通过 API 密钥使用，或通过 ACP 使用现有的 Claude、ChatGPT 或 Gemini 订阅。此外，它还通过模型上下文协议 (MCP) 开放标准连接了 70 多种扩展插件。

**goose is part of the Agentic AI Foundation (AAIF) at the Linux Foundation.**

goose 是 Linux 基金会旗下智能体 AI 基金会 (AAIF) 的一部分。
