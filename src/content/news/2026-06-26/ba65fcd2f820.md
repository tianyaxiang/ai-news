# 厌倦了枯燥、刻板的网页设计？试试这个有趣的 CSS 框架！🎨

Let’s be honest: modern web design can feel a little repetitive. Everything is a perfectly sharp, clean, corporate box built with Tailwind or Bootstrap. If you want to make something that stands out and feels alive, you need Butterfly CSS. 🦋
说实话：现代网页设计有时会让人感到有些千篇一律。一切都是用 Tailwind 或 Bootstrap 构建的、棱角分明且整洁的“企业风”方块。如果你想制作一些脱颖而出且充满活力的内容，你需要 Butterfly CSS。🦋

What is it? Butterfly CSS is an attribute-based CSS framework designed to give your websites a unique, playful, hand-drawn look with absolute zero effort.
它是什么？Butterfly CSS 是一个基于属性（attribute-based）的 CSS 框架，旨在让你无需费力即可为网站赋予独特、俏皮且手绘风格的外观。

Why you'll love it:
你为什么会喜欢它：

Attribute-Based Simplicity: No massive string of utility classes. You just add straightforward HTML attributes. Want a button to look sketchy? Add `handdrawn`. Want it turn into flying butterfly? Add `butterfly='fly'`.
基于属性的简洁性：无需冗长的工具类字符串。你只需添加直观的 HTML 属性即可。想让按钮看起来像手绘的？添加 `handdrawn`。想让它变成飞舞的蝴蝶？添加 `butterfly='fly'`。

Unique Style: It instantly injects personality into your project, making it look like a creative sketch while maintaining perfect responsive layouts.
独特的风格：它能瞬间为你的项目注入个性，使其看起来像是一幅创意草图，同时保持完美的响应式布局。

Built-in Power: It comes with easy dark-mode handling, flexible flexbox grids, and a beautiful color palette out of the box.
内置功能强大：它开箱即用，支持便捷的深色模式处理、灵活的 Flexbox 网格以及精美的配色方案。

Perfect for portfolios, indie games, or indie hacker side-projects that need to catch a user's eye instantly.
非常适合需要瞬间抓住用户眼球的作品集、独立游戏或独立开发者的副业项目。
