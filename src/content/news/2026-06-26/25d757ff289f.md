# Chorus II: Cross-Request Sparsity Reuse for Efficient Image-to-Video Generation
# Chorus II：用于高效图生视频生成的跨请求稀疏性重用

**Abstract:** Serving diffusion models for image-to-video generation is computationally expensive, posing significant challenges for large-scale deployment. Real I2V workloads often contain similar requests, such as repeated effect templates, related subjects, and recurring shot layouts. Existing cross-request acceleration methods mainly exploit this redundancy through feature reuse.

**摘要：** 为图生视频（I2V）生成任务提供扩散模型服务计算成本高昂，这对大规模部署构成了重大挑战。实际的 I2V 工作负载通常包含相似的请求，例如重复的特效模板、相关的主体以及循环出现的镜头布局。现有的跨请求加速方法主要通过特征重用来利用这种冗余。

We observe that similar I2V requests also share highly consistent sparse attention patterns, enabling historical sparse masks to serve as request-conditioned priors with almost no online mask-prediction overhead. We propose a cross-request reuse framework centered on **sparsity reuse**, with **feature reuse** as an optional extension safeguarded by a lightweight **guidance enhancement**.

我们观察到，相似的 I2V 请求也共享高度一致的稀疏注意力模式，这使得历史稀疏掩码（sparse masks）能够作为请求条件先验，且几乎没有在线掩码预测的开销。我们提出了一个以**稀疏性重用（sparsity reuse）**为核心的跨请求重用框架，并将**特征重用（feature reuse）**作为可选扩展，同时辅以轻量级的**引导增强（guidance enhancement）**来保障效果。

Our sparsity reuse is implemented as shared sparse mask reuse, which reuses high-quality sparse masks from similar historical requests to avoid per-request online mask prediction. Optional feature reuse applies downsampled computation to highly redundant spatiotemporal regions, mitigating boundary artifacts while preserving efficiency gains. Guidance enhancement reinforces image/text conditioning after reuse, mitigating semantic drift and condition-adherence issues.

我们的稀疏性重用实现为共享稀疏掩码重用，即通过重用来自相似历史请求的高质量稀疏掩码，从而避免了每个请求的在线掩码预测。可选的特征重用将降采样计算应用于高度冗余的时空区域，在保持效率提升的同时减轻了边界伪影。引导增强在重用后强化了图像/文本的条件约束，缓解了语义漂移和条件依从性问题。

Experiments show that default sparsity reuse configuration preserves generation quality with a **2.16×** speedup.

实验表明，默认的稀疏性重用配置在保持生成质量的同时，实现了 **2.16 倍**的加速。
