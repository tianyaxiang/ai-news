# Wan-Streamer v0.1: End-to-end Real-time Interactive Foundation Models
# Wan-Streamer v0.1：端到端实时交互基础模型

We present Wan-Streamer, a native-streaming, end-to-end interactive foundation model designed from the ground up for real-time, low-latency, full-duplex audio-visual interaction. 
我们推出了 Wan-Streamer，这是一个原生流式、端到端的交互式基础模型，专为实时、低延迟、全双工的视听交互而从零设计。

Wan-Streamer seamlessly models language, audio, and video as both input and output within a single Transformer, where the sequence is represented as interleaved visual, audio, and text input tokens together with visual, audio, and text output tokens, coordinated by block-causal attention for incremental streaming. 
Wan-Streamer 在单个 Transformer 中无缝地将语言、音频和视频建模为输入和输出。序列以交错的视觉、音频和文本输入 Token 以及视觉、音频和文本输出 Token 的形式呈现，并通过块因果注意力（block-causal attention）进行协调，以实现增量式流处理。

Unlike cascaded interactive systems that rely on separate VAD, ASR, language, TTS, audio-driven animation, or video-generation modules, Wan-Streamer does not rely on external language, speech, avatar, or video-generation modules: perception, reasoning, generation, response timing, turn management, and cross-modal synchronization are learned jointly within one unified model, reducing pipeline latency and error accumulation. 
与依赖独立的 VAD（语音活动检测）、ASR（自动语音识别）、语言、TTS（语音合成）、音频驱动动画或视频生成模块的级联交互系统不同，Wan-Streamer 不依赖外部的语言、语音、虚拟形象或视频生成模块：感知、推理、生成、响应时机、轮次管理和跨模态同步都在一个统一的模型中联合学习，从而减少了流水线延迟和误差累积。

To support natural audio-visual responsiveness, we redesign the entire stack around streamability, including causal encoders, causal decoders, block-causal attention, and low-latency multimodal token scheduling, enabling streaming units as short as 160 ms at 25 fps. 
为了支持自然的视听响应能力，我们围绕流式传输特性重新设计了整个技术栈，包括因果编码器、因果解码器、块因果注意力和低延迟多模态 Token 调度，实现了在 25 fps 下短至 160 毫秒的流式处理单元。

Wan-Streamer achieves approximately 200 ms model-side response latency and approximately 550 ms total interaction latency when combined with 350 ms bidirectional network latency, supporting sub-second duplex audio-visual communication. These results position Wan-Streamer as a unified, end-to-end, multimodal interactive foundation model for low-latency streaming interaction.
Wan-Streamer 实现了约 200 毫秒的模型端响应延迟；在结合 350 毫秒的双向网络延迟后，总交互延迟约为 550 毫秒，支持亚秒级的双工视听通信。这些结果表明，Wan-Streamer 是一个用于低延迟流式交互的统一、端到端的多模态交互基础模型。
