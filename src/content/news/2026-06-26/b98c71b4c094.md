# AgentOdyssey: Open-Ended Long-Horizon Text Game Generation for Test-Time Continual Learning Agents
# AgentOdyssey：面向测试时持续学习智能体的开放式长程文本游戏生成框架

**Abstract:** For agents to learn continuously from interaction with the world at test time, they must be able to explore effectively, acquire new world knowledge and skills, retain relevant episodic experiences, and plan over long horizons. To evaluate these key abilities of test-time continual learning agents, we introduce AgentOdyssey, a novel evaluation framework that procedurally generates open-ended text games with rich entities, world dynamics, and long-horizon tasks.

**摘要：** 为了使智能体能够在测试时通过与世界的交互进行持续学习，它们必须具备有效探索、获取新世界知识与技能、保留相关情景经验以及进行长程规划的能力。为了评估测试时持续学习智能体的这些关键能力，我们引入了 AgentOdyssey，这是一个全新的评估框架，它能够程序化地生成包含丰富实体、世界动态和长程任务的开放式文本游戏。

Critically, AgentOdyssey goes beyond the conventional machine learning assumption that learning does not occur at test time by placing agents in a continuous, long-horizon setting that interleaves learning and inference throughout deployment. We further propose a multifaceted evaluation methodology that measures not only game progress but also offers diagnostic tests on world knowledge acquisition, episodic memory, object and action exploration, action diversity, and model cost.

至关重要的是，AgentOdyssey 超越了传统机器学习中“测试时不进行学习”的假设，将智能体置于一个持续的、长程的环境中，使其在整个部署过程中交替进行学习与推理。我们进一步提出了一种多维度的评估方法，不仅衡量游戏进度，还针对世界知识获取、情景记忆、物体与动作探索、动作多样性以及模型成本提供诊断测试。

We evaluate diverse agent paradigms in the generated games. Our experimental results reveal critical limits in agents' key abilities, as well as factors that influence their meaningful horizon. Although performance scales with stronger base models, even the top agent remains far below human performance, leaving substantial headroom for improvement. Among agent mechanisms, we find that short-term memory benefits multiple agent paradigms and is an important component of agent test-time training.

我们在生成的游戏中评估了多种智能体范式。实验结果揭示了智能体在关键能力上的局限性，以及影响其有效视野（meaningful horizon）的因素。尽管性能会随着基础模型的增强而提升，但即使是表现最好的智能体，其水平仍远低于人类，仍有巨大的提升空间。在智能体机制方面，我们发现短期记忆对多种智能体范式都有益处，是智能体测试时训练的重要组成部分。
