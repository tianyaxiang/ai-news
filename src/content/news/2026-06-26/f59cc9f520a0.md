# OS9Map: OpenStreetMap for Mac OS 9
# OS9Map：适用于 Mac OS 9 的 OpenStreetMap

OS9Map lets you browse OpenStreetMap on Mac OS 9. Search for landmarks and addresses, and save the places you care about as bookmarks for next time.
OS9Map 让您能够在 Mac OS 9 上浏览 OpenStreetMap。您可以搜索地标和地址，并将您关注的地点保存为书签，以便下次快速访问。

### Requirements
### 系统要求

*   Mac OS 9
*   PowerPC processor
*   PowerPC 处理器
*   16 MB RAM (32 MB or more recommended)
*   16 MB 内存（建议 32 MB 或以上）
*   An internet connection (Open Transport TCP/IP)
*   互联网连接（Open Transport TCP/IP）

### Features
### 功能特性

*   **Smooth scrolling map canvas** — drag with the mouse to pan; nearby tiles load as you go.
*   **平滑滚动地图画布** — 使用鼠标拖动即可平移；地图瓦片会随您的移动自动加载。
*   **Place search** — built-in Nominatim lookup; type an address to jump straight there.
*   **地点搜索** — 内置 Nominatim 查询功能；输入地址即可直接跳转。
*   **Bookmarks** — save the places you visit often and return to them from the menu in one click.
*   **书签** — 保存您常去的地点，并通过菜单一键返回。
