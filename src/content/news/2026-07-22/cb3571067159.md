# ForensicNet: Lightweight Attention-Enhanced MobileNetV2 for Automated Face Identification
# ForensicNet：用于自动人脸识别的轻量级注意力增强型 MobileNetV2

**Abstract:** In forensic environments, automated identification of perpetrators is difficult due to pose changes, changes in light, occlusion, and lack of labeled data. This paper presents ForensicNet, a lightweight deep learning framework for forensic face recognition that enhances attention.
**摘要：** 在法医鉴定环境中，由于姿态变化、光照改变、遮挡以及标记数据匮乏，自动识别犯罪嫌疑人面临诸多困难。本文提出了 ForensicNet，这是一个用于法医人脸识别的轻量级深度学习框架，通过增强注意力机制来解决上述问题。

The suggested model combines the MobileNetV2 backbone with Convolutional Block Attention Modules (CBAM) to improve the learning of discriminative features while maintaining computational speed. A two-phase transfer learning strategy with adaptive layer unfreezing is used to improve domain adaptation and reduce overfitting.
该模型将 MobileNetV2 主干网络与卷积块注意力模块（CBAM）相结合，在保持计算速度的同时，提升了对判别性特征的学习能力。研究采用了一种带有自适应层解冻的两阶段迁移学习策略，以改善领域适应性并减少过拟合。

This study used publicly available datasets such as LFW and SCFace, with 15,000 facial images spanning 68 identity classes. The proposed model outperforms baseline architectures such as AlexNet, ResNet-50, and MobileNetV2, with an accuracy of 92.4%, a precision of 90.8%, and a recall of 89.5%.
本研究使用了 LFW 和 SCFace 等公开数据集，涵盖了 68 个身份类别的 15,000 张人脸图像。所提出的模型在性能上优于 AlexNet、ResNet-50 和 MobileNetV2 等基准架构，其准确率达到 92.4%，精确率为 90.8%，召回率为 89.5%。

Additionally, the framework requires only 2.1 GFLOPs per inference, and hence can be used in real-time forensic surveillance applications.
此外，该框架每次推理仅需 2.1 GFLOPs 的计算量，因此可应用于实时法医监控场景。
