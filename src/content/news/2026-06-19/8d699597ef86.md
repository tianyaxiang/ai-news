# Locofy: Design-to-Code Agents

**Locofy: design-to-code agents**
Locofy：设计转代码智能体

**Go from designs to developer-friendly frontend code in a flash without changing your tech stack or workflows. Get responsive, interactive, and modular code that you can easily export or sync to GitHub/VS Code.**
无需更改技术栈或工作流程，即可快速将设计稿转换为开发者友好的前端代码。获取响应式、交互式且模块化的代码，并可轻松导出或同步至 GitHub/VS Code。

**Locofy is the agentic frontend layer between Figma and Cursor & Claude. Using proprietary design-aware models combined with LLMs, Locofy converts Figma designs into structured, responsive, pixel-perfect frontend code and works directly from the CLI, Cursor, Claude Code, and others.**
Locofy 是连接 Figma 与 Cursor 及 Claude 之间的智能前端层。通过结合专有的“设计感知”模型与大语言模型（LLM），Locofy 能将 Figma 设计稿转换为结构化、响应式且像素级精准的前端代码，并可直接在 CLI、Cursor、Claude Code 等工具中运行。

**While coding agents excel at logic and iteration, they still struggle with UI precision and Figma structure — Locofy provides the frontend foundation they can reliably build on.**
虽然编程智能体在逻辑处理和迭代方面表现出色，但在 UI 精确度和 Figma 结构理解上仍存在困难——而 Locofy 正是为它们提供了可信赖的前端构建基础。
