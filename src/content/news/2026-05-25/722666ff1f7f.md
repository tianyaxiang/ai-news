# Microsoft open-sources "the earliest DOS source code discovered to date"
# 微软开源“迄今为止发现的最早 DOS 源代码”

Several times in the last couple of decades, Microsoft has released source code for the original MS-DOS operating system that kicked off its decades-long dominance of consumer PCs. This week, the company has reached further back than ever, releasing “the earliest DOS source code discovered to date” along with other documentation and notes from its developer.

在过去的几十年里，微软曾多次发布原始 MS-DOS 操作系统的源代码，正是该系统开启了微软在个人电脑领域长达数十年的统治地位。本周，微软追溯到了比以往更早的时期，发布了“迄今为止发现的最早 DOS 源代码”，以及来自其开发者的其他文档和笔记。

Today’s source release is so old that it predates the MS-DOS branding, and it includes “sources to the 86-DOS 1.00 kernel, several development snapshots of the PC-DOS 1.00 kernel, and some well-known utilities such as CHKDSK,” write Microsoft’s Stacey Haffner and Scott Hanselman in their co-authored post about the release.

微软的 Stacey Haffner 和 Scott Hanselman 在关于此次发布的联合博文中写道，今天发布的源代码年代久远，甚至早于“MS-DOS”这一品牌名称。其中包括“86-DOS 1.00 内核的源代码、PC-DOS 1.00 内核的几个开发快照，以及一些众所周知的实用程序，如 CHKDSK”。

To understand the context, here’s a very brief history of what would become MS-DOS: Programmer Tim Paterson originally created 86-DOS (previously known as QDOS, for “quick and dirty operating system”) for an Intel 8086-based computer kit sold by Seattle Computer Products. Microsoft, on the hook to provide an operating system for the still-in-development IBM PC 5150, licensed 86-DOS and hired Paterson to continue developing it, later buying the rights to 86-DOS outright.

为了了解背景，以下是 MS-DOS 演变史的简要回顾：程序员 Tim Paterson 最初为 Seattle Computer Products 出售的基于 Intel 8086 的计算机套件创建了 86-DOS（前身为 QDOS，意为“快速且粗糙的操作系统”）。当时，微软正负责为尚在开发中的 IBM PC 5150 提供操作系统，于是授权使用了 86-DOS，并聘请 Paterson 继续开发，随后彻底买下了 86-DOS 的版权。

Microsoft then licensed this operating system to IBM as PC-DOS while retaining the ability to sell the operating system to other companies. The version sold by Microsoft was called MS-DOS, and the proliferation of third-party IBM PC clones over the ’80s and ’90s made it the version of the operating system that most people ended up using.

随后，微软将该操作系统授权给 IBM，命名为 PC-DOS，同时保留了向其他公司销售该操作系统的权利。微软销售的版本被称为 MS-DOS，随着 80 年代和 90 年代第三方 IBM PC 兼容机的激增，它最终成为了大多数人使用的操作系统版本。

This source code is old enough that it hadn’t been stored digitally. “A dedicated team of historians and preservationists led by Yufeng Gao and Rich Cini,” calling itself the “DOS Disassembly Group,” painstakingly transcribed and scanned in code from paper printouts provided by Paterson. This process was made even more difficult because modern OCR software struggled with the quality of the decades-old printout.

这些源代码年代久远，当时并未以数字形式存储。一个由 Yufeng Gao 和 Rich Cini 领导的“历史学家和保护主义者团队”，自称为“DOS 反汇编小组”（DOS Disassembly Group），从 Paterson 提供的纸质打印件中辛苦地转录并扫描了这些代码。由于现代 OCR 软件难以处理这些几十年前的打印件质量，这一过程变得更加困难。

Microsoft has also open-sourced several of its other early software projects. In 2014 (and again in 2018), the company open-sourced MS-DOS versions 1.25 and 2.0. It followed that up in 2024 with the oddball MS-DOS 4.0 release. Those versions are all available in the same GitHub repo. Other open-sourced projects include the game Zork and its sequels and 1995’s Microsoft 3D Movie Maker (plans to modernize this app and add new features have largely gone nowhere).

微软此前还开源了其他几个早期软件项目。2014 年（以及 2018 年），该公司开源了 MS-DOS 1.25 和 2.0 版本。随后在 2024 年，又发布了较为特殊的 MS-DOS 4.0 版本。这些版本都可以在同一个 GitHub 仓库中找到。其他开源项目还包括游戏《魔域》（Zork）及其续作，以及 1995 年的《微软 3D 电影制作大师》（Microsoft 3D Movie Maker，但对其进行现代化改造并增加新功能的计划基本已搁浅）。

The open source remake of the old MS-DOS Editor isn’t actually the same app as the old EDIT.COM, but its heart is in the right place. For students of early PC history, this isn’t even the first piece of 86-DOS history that has been newly rediscovered this decade. Just two years ago, the earliest known version of 86-DOS was rediscovered and uploaded to the Internet Archive.

虽然旧版 MS-DOS 编辑器的开源重制版实际上与旧的 EDIT.COM 并非同一个应用程序，但其初衷是好的。对于早期 PC 历史的研究者来说，这甚至不是本十年内重新发现的第一段 86-DOS 历史。就在两年前，已知最早的 86-DOS 版本也被重新发现并上传到了互联网档案馆（Internet Archive）。
