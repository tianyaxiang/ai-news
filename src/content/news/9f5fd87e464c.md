# FruitProM-V2: Robust Probabilistic Maturity Estimation and Detection of Fruits and Vegetables
# FruitProM-V2：果蔬鲁棒概率成熟度评估与检测

**Abstract:** Accurate fruit maturity identification is essential for determining harvest timing, as incorrect assessment directly affects yield and post-harvest quality.
**摘要：** 准确识别果实成熟度对于确定采收时间至关重要，因为评估错误会直接影响产量和采后品质。

Although ripening is a continuous biological process, vision-based maturity estimation is typically formulated as a multi-class classification task, which imposes sharp boundaries between visually similar stages.
尽管成熟是一个连续的生物过程，但基于视觉的成熟度评估通常被表述为多分类任务，这在视觉上相似的阶段之间强加了明确的界限。

To examine this limitation, we perform an annotation reliability study with two independent annotators on a held-out tomato dataset and observe disagreement concentrated near adjacent maturity stages.
为了检验这一局限性，我们对一个留出的番茄数据集进行了标注可靠性研究，由两名独立的标注员进行标注，并观察到分歧主要集中在相邻的成熟阶段附近。

Motivated by this observation, we model maturity as a latent continuous variable and predict it probabilistically using a distributional detection head, converting the distribution into class probabilities through the cumulative distribution function (CDF).
受此观察的启发，我们将成熟度建模为一个潜在的连续变量，并使用分布检测头对其进行概率预测，通过累积分布函数（CDF）将分布转换为类别概率。

The proposed formulation maintains comparable performance to a standard detector under clean labels while better representing uncertainty.
所提出的公式在干净标签下保持了与标准检测器相当的性能，同时能更好地表示不确定性。

Furthermore, when controlled label noise is introduced during training, the probabilistic model demonstrates improved robustness relative to the baseline, indicating that explicitly modeling maturity uncertainty leads to more reliable visual maturity estimation.
此外，当在训练过程中引入受控的标签噪声时，该概率模型相对于基线表现出了更强的鲁棒性，这表明显式地对成熟度不确定性进行建模可以实现更可靠的视觉成熟度评估。
