# From Confident Closing to Silent Failure: Characterizing False Success in LLM Agents
# 从自信结项到静默失败：LLM 智能体中“虚假成功”现象的特征分析

**Abstract:** LLM agents can fail silently by asserting task completion when the environment state shows otherwise. We study this failure mode, false success, across two agent benchmarks: 9,876 tau2-bench trajectories from 8 model families and 1,879 AppWorld trajectories from 4 model families with text-independent ground truth.

**摘要：** 大语言模型（LLM）智能体可能会在环境状态显示任务未完成时，却断言任务已完成，从而导致“静默失败”。我们针对这种被称为“虚假成功”（false success）的失败模式进行了研究，涵盖了两个智能体基准测试：包含来自 8 个模型家族的 9,876 条 tau2-bench 轨迹，以及包含来自 4 个模型家族、具有文本无关事实依据（ground truth）的 1,879 条 AppWorld 轨迹。

False success is common but varies by setting: 45--48% of failures in single-control tau2-bench domains, 3% in dual-control telecom, and 75.8% among AppWorld self-assessing coding-agent trajectories with explicit status claims. LLM judges fail reliably: no configuration across 5 judges, 5 prompt strategies, and full task specifications exceeds AUROC 0.65 on tau2-bench, and the same judges reach only 0.54 AUROC on AppWorld API-call traces.

“虚假成功”现象十分普遍，但因场景而异：在单控制 tau2-bench 领域中，失败案例的 45%–48% 为虚假成功；在双控制电信领域中为 3%；而在具有明确状态声明的 AppWorld 自评估编码智能体轨迹中，这一比例高达 75.8%。LLM 裁判的表现同样不可靠：在 5 种裁判模型、5 种提示策略以及完整任务规范的各种配置下，其在 tau2-bench 上的 AUROC 指标均未超过 0.65，而在 AppWorld API 调用轨迹上，这些裁判的 AUROC 仅达到 0.54。

Judges rely on surface completion proxies -- confident closing language in tau2-bench and coarse action-sequence volume in AppWorld -- rather than verified state changes. Lightweight TF-IDF detectors achieve task-disjoint AUROC 0.83 on tau2-bench and 0.95 on AppWorld, recovering 4--8x more false successes than the best judge at the same flag rate with 3,300x lower latency.

这些裁判依赖于表面的完成代理指标——例如 tau2-bench 中的“自信结项语言”和 AppWorld 中的“粗略动作序列数量”——而非验证实际的状态变更。相比之下，轻量级的 TF-IDF 检测器在 tau2-bench 和 AppWorld 上分别实现了 0.83 和 0.95 的任务无关 AUROC。在相同的标记率下，它们发现“虚假成功”的能力是表现最好的裁判的 4 到 8 倍，且延迟降低了 3,300 倍。

These results suggest that production monitoring should use lightweight, domain-calibrated detectors as triage signals rather than relying on LLM judges as the primary monitor for false success.

这些研究结果表明，在生产环境监控中，应优先使用轻量级、经过领域校准的检测器作为分流信号，而不是依赖 LLM 裁判作为监控“虚假成功”的主要手段。
