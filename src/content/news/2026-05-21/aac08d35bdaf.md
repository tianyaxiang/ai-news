### Glia

**Glia**

Launching today, Glia is a local-first AI memory bridge between browser chats and IDEs.
今天正式发布，Glia 是一个连接浏览器聊天记录与集成开发环境（IDE）的本地优先 AI 记忆桥梁。

Glia is a 100% offline, open-source memory bridge. A Chrome extension auto-saves your web-based Claude/ChatGPT chats, while a native MCP server lets Cursor/Claude Code query those decisions locally from your shared SQLite database.
Glia 是一个 100% 离线、开源的记忆桥梁。通过一个 Chrome 扩展程序，它可以自动保存你在网页端 Claude 或 ChatGPT 的聊天记录；同时，其原生 MCP 服务器允许 Cursor 或 Claude Code 从你共享的 SQLite 数据库中本地查询这些决策信息。
