# Simply Stabilizing the Loop via Fully Looped Transformer
# 通过全循环 Transformer 实现循环的简单稳定化

Scaling model performance typically requires increasing model size. Looped Transformer offers a compelling alternative by iteratively reusing the same Transformer blocks, trading additional computation for improved performance without increasing parameter count or context length.
扩展模型性能通常需要增加模型规模。循环 Transformer（Looped Transformer）提供了一种极具吸引力的替代方案，它通过迭代重复使用相同的 Transformer 块，以额外的计算量换取性能提升，且无需增加参数量或上下文长度。

Because the number of loop iterations can be adjusted at inference, it also provides a natural mechanism for balancing performance and test-time compute. However, Looped Transformer still suffers from training instability when the number of loop iterations increases.
由于循环迭代次数可以在推理阶段进行调整，它还提供了一种平衡性能与测试时计算量的自然机制。然而，当循环迭代次数增加时，循环 Transformer 仍然会面临训练不稳定的问题。

Our analysis reveals that this instability stems from two sources: gradient oscillation and residual explosion. To address these two problems, we propose the Fully Looped Transformer, which introduces two parameter-free modifications: (1) Fully Looped Architecture, which distributes inter-loop signals across all layers to mitigate residual explosion; (2) Attention Injection, which reuses the existing attention block to suppress gradient oscillation.
我们的分析表明，这种不稳定性源于两个方面：梯度震荡和残差爆炸。为了解决这两个问题，我们提出了全循环 Transformer（Fully Looped Transformer），并引入了两种无需额外参数的改进：(1) 全循环架构，通过将循环间的信号分布到所有层来缓解残差爆炸；(2) 注意力注入，通过复用现有的注意力块来抑制梯度震荡。

These modifications stabilize training dynamics, enabling the Fully Looped Transformer to be trained stably up to 12 loop iterations, whereas other baseline looped models collapse in this regime. In milder settings where Looped Transformer does not collapse, Fully Looped Transformer still improves average downstream-task performance by up to 13.2%.
这些改进稳定了训练动态，使得全循环 Transformer 能够稳定训练至多达 12 次循环迭代，而其他基准循环模型在此设置下会崩溃。在循环 Transformer 不会崩溃的较温和设置下，全循环 Transformer 仍能将下游任务的平均性能提升高达 13.2%。

Overall, our experiments demonstrate that Fully Looped Transformer improves training stability, enhances downstream performance, and provides preliminary adaptability under different test-time compute budgets by varying loop iterations at inference.
总的来说，我们的实验证明，全循环 Transformer 改善了训练稳定性，增强了下游性能，并通过在推理时改变循环迭代次数，为不同的测试时计算预算提供了初步的适应性。
