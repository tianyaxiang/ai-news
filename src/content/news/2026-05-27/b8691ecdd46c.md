### DNSimple CLI

**DNSimple CLI**

Whether you manage one domain or 100,000 domains, DNSimple delivers simple, secure domain management solutions.
无论您是管理一个域名还是十万个域名，DNSimple 都能为您提供简单、安全的域名管理解决方案。

**Manage Your DNS from the Command Line with DNSimple CLI**
**使用 DNSimple CLI 从命令行管理您的 DNS**

Manage your domains and DNS without leaving the terminal, built for developers who live in their terminals and CI pipelines.
无需离开终端即可管理您的域名和 DNS，专为那些习惯在终端和 CI 流水线中工作的开发者打造。

What you can do: create, update, and delete DNS records; register domains and manage renewals; issue and manage SSL/TLS certificates; run it in CI with API token auth and structured output.
您可以实现的功能包括：创建、更新和删除 DNS 记录；注册域名并管理续费；签发和管理 SSL/TLS 证书；通过 API 令牌认证和结构化输出在 CI 环境中运行。

Built in Go, it uses the same foundation as our official client libraries and Terraform provider.
该工具使用 Go 语言构建，与我们的官方客户端库和 Terraform 提供程序基于相同的底层架构。
