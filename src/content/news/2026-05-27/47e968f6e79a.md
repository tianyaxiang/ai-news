### Parsewise API

**Parsewise API for agentic multi-document processing**
Parsewise API：面向智能体（Agentic）的多文档处理接口

**With a single call, you can provide multiple input documents, a desired output, and get back a response with resolved values, lineage across documents / pages, contradiction detection and bounding boxes that you can display in your own app. No need to duct tape together parsing, classification, extractions, resolutions, and custom verification UIs.**
只需一次调用，您即可提供多个输入文档及预期的输出格式，并获得包含解析值、跨文档/页面溯源、矛盾检测以及可直接在您的应用中显示的边界框（bounding boxes）的响应。无需再费力拼凑解析、分类、提取、解析和自定义验证界面。

**One API call replaces the entire document processing pipeline. You send multiple documents and a desired output schema; you get back resolved values, flagged contradictions, and full lineage down to the source words, pages, and documents, with bounding boxes you can embed directly into your own UI for human validation. No parsing, classification, stitching, or custom verification interface to build or maintain.**
一次 API 调用即可替代整个文档处理流程。您只需发送多个文档和预期的输出模式（schema），即可获得解析后的数值、标记出的矛盾点，以及精确到源词、页面和文档的完整溯源信息，并附带可直接嵌入您 UI 以供人工验证的边界框。无需构建或维护任何解析、分类、拼接或自定义验证界面。
