# GitHub Actions was down
# GitHub Actions 曾发生故障

GitHub Actions experienced a major service outage, impacting users' ability to run workflows and automation tasks.
GitHub Actions 曾经历了一次重大服务中断，影响了用户运行工作流和自动化任务的能力。

The incident, which occurred recently, resulted in a temporary disruption of various GitHub services. The status page indicated that while most systems remained operational, the Actions component faced significant performance issues.
此次事故发生于近期，导致了 GitHub 多项服务的暂时性中断。状态页面显示，尽管大多数系统保持正常运行，但 Actions 组件面临了严重的性能问题。

According to the official status report, the incident has since been resolved. GitHub expressed gratitude to its users for their patience during the downtime.
根据官方状态报告，该事故现已得到解决。GitHub 对用户在停机期间的耐心等待表示感谢。

For ongoing monitoring, users can check the status of specific regions, including Australia, EU, Japan, and the US, via the dedicated GitHub status portals.
为了进行持续监控，用户可以通过专门的 GitHub 状态门户网站查看包括澳大利亚、欧盟、日本和美国在内的特定区域的服务状态。

Historical uptime data for various services, including Git Operations, Webhooks, API Requests, and Codespaces, remains available on the official status website for transparency.
为了保持透明度，包括 Git 操作、Webhooks、API 请求和 Codespaces 在内的各项服务的历史在线时间数据，均可在官方状态网站上查阅。
